# VSCode 설치 동영상 제작 가이드 (하이브리드 방식)

## 개요

인트로/아웃트로는 Vrew AI 생성 영상을 사용하고, 본문은 실제 화면녹화를 활용하는 하이브리드 방식입니다.

---

## 전체 구조

```
┌─────────────────────────────────────────────────────────┐
│  [인트로] AI 생성 영상 (30초~1분)                        │
│  - Vrew "텍스트로 비디오 만들기" 활용                    │
│  - 영상 주제 소개, 시청자 관심 유도                      │
├─────────────────────────────────────────────────────────┤
│  [본문] 실제 화면녹화 + AI 나레이션 (5~10분)            │
│  - Windows 화면녹화 (OBS, Windows 게임바 등)            │
│  - Vrew에서 AI 음성 나레이션 추가                       │
│  - 자동 자막 생성                                       │
├─────────────────────────────────────────────────────────┤
│  [아웃트로] AI 생성 영상 (30초~1분)                      │
│  - 요약 및 다음 영상 예고                               │
│  - 구독/좋아요 유도                                     │
└─────────────────────────────────────────────────────────┘
```

---

## 단계별 제작 과정

### Phase 1: 사전 준비

#### 1.1 시나리오 작성
- 인트로 대본 (텍스트→비디오용)
- 본문 나레이션 대본 (화면녹화용)
- 아웃트로 대본 (텍스트→비디오용)

#### 1.2 필요 도구 준비
| 도구 | 용도 | 비고 |
|------|------|------|
| Vrew | AI 영상 생성, 나레이션, 자막 | 메인 편집 도구 |
| OBS Studio | 화면녹화 | 무료, 고품질 |
| Windows 게임바 | 화면녹화 (대안) | Win+G로 실행 |

---

### Phase 2: 인트로 제작 (Vrew 텍스트→비디오)

#### 2.1 Vrew에서 새 프로젝트 생성
1. Vrew 실행 → "텍스트로 비디오 만들기" 선택
2. 인트로 대본 입력

#### 2.2 인트로 대본 예시
```
안녕하세요, 오늘은 Windows에서 VS Code를 설치하는 방법을 알려드리겠습니다.

VS Code는 마이크로소프트에서 만든 무료 코드 편집기로,
전 세계 개발자들이 가장 많이 사용하는 도구입니다.

이 영상을 따라하시면 5분 안에 VS Code 설치를 완료하실 수 있습니다.
지금 바로 시작해볼까요?
```

#### 2.3 AI 영상 스타일 설정
- 음성: 한국어 남성/여성 선택
- 배경: 기술/코딩 관련 이미지
- 자막 스타일: 깔끔한 폰트

---

### Phase 3: 본문 제작 (화면녹화 + AI 나레이션)

#### 3.1 화면녹화 준비
1. **OBS Studio 설정**
   - 해상도: 1920x1080 (Full HD)
   - 프레임: 30fps
   - 오디오: 음소거 (나레이션은 나중에 추가)

2. **녹화할 장면 목록**
   | Scene | 내용 | 예상 시간 |
   |-------|------|----------|
   | 1 | VS Code 공식 사이트 접속 | 30초 |
   | 2 | 다운로드 버튼 클릭 | 20초 |
   | 3 | 설치 파일 실행 | 30초 |
   | 4 | 설치 옵션 선택 | 1분 |
   | 5 | 설치 완료 및 실행 | 30초 |
   | 6 | 기본 설정 (한글 확장 등) | 2분 |

#### 3.2 화면녹화 진행
1. OBS 녹화 시작
2. 각 Scene별로 천천히, 명확하게 진행
3. 실수 시 잠시 멈추고 다시 진행 (편집으로 제거)

#### 3.3 Vrew에서 나레이션 추가
1. 녹화된 영상을 Vrew에 불러오기
2. "AI 음성 만들기" 기능 사용
3. Scene별 나레이션 대본 입력

#### 3.4 본문 나레이션 대본 예시
```
[Scene 1]
먼저 웹 브라우저를 열고 VS Code 공식 사이트에 접속합니다.
주소창에 code.visualstudio.com을 입력하세요.

[Scene 2]
사이트에 접속하면 다운로드 버튼이 보입니다.
Windows용 다운로드 버튼을 클릭하세요.

[Scene 3]
다운로드가 완료되면 설치 파일을 실행합니다.
화면 하단의 다운로드 파일을 클릭하거나,
다운로드 폴더에서 직접 실행할 수 있습니다.

[Scene 4]
설치 마법사가 나타납니다.
라이선스 동의에 체크하고 다음을 클릭하세요.

여기서 중요한 옵션이 있습니다.
"PATH에 추가" 옵션을 반드시 체크해주세요.
이 옵션을 체크하면 터미널에서 code 명령어로 VS Code를 실행할 수 있습니다.

[Scene 5]
설치가 완료되었습니다.
"VS Code 실행" 체크박스를 선택하고 마침을 클릭하세요.

[Scene 6]
VS Code가 실행되었습니다.
한글 사용을 위해 Korean Language Pack을 설치해보겠습니다.
왼쪽 확장 아이콘을 클릭하고, Korean을 검색합니다.
설치 버튼을 클릭하면 한글 패치가 완료됩니다.
```

#### 3.5 자막 생성
- Vrew 자동 자막 기능 활용
- 생성된 자막 검토 및 수정

---

### Phase 4: 아웃트로 제작 (Vrew 텍스트→비디오)

#### 4.1 아웃트로 대본 예시
```
오늘 VS Code 설치 방법을 알아보았습니다.

다음 영상에서는 VS Code에서 첫 번째 프로그램을 작성하는 방법을
알려드리겠습니다.

영상이 도움이 되셨다면 좋아요와 구독 부탁드립니다.
궁금한 점은 댓글로 남겨주세요.

감사합니다!
```

#### 4.2 엔딩 요소 추가
- 구독 버튼 애니메이션
- 다음 영상 썸네일 미리보기
- 채널 로고

---

### Phase 5: 최종 편집 및 병합

#### 5.1 Vrew에서 전체 병합
1. 인트로 영상 가져오기
2. 본문 영상 연결
3. 아웃트로 영상 연결

#### 5.2 전환 효과 추가
- 각 파트 사이에 페이드 인/아웃 효과
- 장면 전환 시 부드러운 효과

#### 5.3 배경음악 추가
- Vrew 제공 무료 BGM 사용
- 볼륨: 나레이션의 10~20% 수준
