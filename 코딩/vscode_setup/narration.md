# VSCode 설치 동영상 나레이션

---

## 인트로

```
안녕하세요!

오늘은 전 세계 개발자들이 가장 많이 사용하는
코드 편집기, VS Code를 설치하는 방법을 알려드리겠습니다.

VS Code는 마이크로소프트에서 만든 무료 프로그램으로,
Python, JavaScript, C언어 등 거의 모든 프로그래밍 언어를 지원합니다.

이 영상 하나면 VS Code 설치부터
기본 설정까지 완벽하게 마칠 수 있습니다.

그럼 지금 바로 시작해볼까요?
```

---

## Scene 1: VS Code 공식 사이트 접속

```
먼저 웹 브라우저를 실행합니다.
크롬이나 엣지 어떤 브라우저를 사용해도 괜찮습니다.

주소창에 code.visualstudio.com을 입력하고 엔터를 누르세요.
이곳이 VS Code 공식 사이트입니다.

화면에 보이는 것처럼 큰 다운로드 버튼이 있습니다.
Windows를 사용하시면 자동으로 Windows용이 표시됩니다.
```

---

## Scene 2: 다운로드

```
다운로드 버튼을 클릭하세요.

파일 크기는 약 90메가바이트 정도로,
인터넷 속도에 따라 1~2분 정도 소요됩니다.

다운로드가 완료될 때까지 잠시 기다려주세요.

다운로드가 완료되면 브라우저 하단에 파일이 표시됩니다.
```

---

## Scene 3: 설치 파일 실행

```
다운로드된 파일을 클릭해서 실행합니다.

윈도우에서 설치를 허용할지 물어보면 "예"를 클릭하세요.

잠시 후 VS Code 설치 마법사가 나타납니다.
```

---

## Scene 4: 라이선스 동의

```
라이선스 계약서 화면입니다.
내용을 확인하시고, "동의합니다"를 선택한 후 다음을 클릭하세요.
```

---

## Scene 5: 설치 위치 선택

```
설치 위치를 선택하는 화면입니다.
특별한 이유가 없다면 기본 경로를 그대로 사용하시면 됩니다.
다음을 클릭하세요.
```

---

## Scene 6: 시작 메뉴 폴더

```
시작 메뉴에 표시될 폴더 이름입니다.
기본값 그대로 다음을 클릭합니다.
```

---

## Scene 7: 추가 작업 선택 (중요!)

```
이 화면이 가장 중요합니다.
추가 옵션을 선택하는 화면인데, 몇 가지 꼭 체크해야 할 것들이 있습니다.

첫 번째, "바탕 화면에 바로 가기 만들기"를 체크하세요.
바탕 화면에서 바로 VS Code를 실행할 수 있어 편리합니다.

두 번째, "PATH에 추가" 옵션입니다.
이 옵션은 반드시 체크해주세요!
이걸 체크하면 명령 프롬프트나 터미널에서
code라는 명령어로 VS Code를 바로 실행할 수 있습니다.
나중에 개발할 때 정말 유용하게 사용됩니다.

세 번째, "Code로 열기" 관련 옵션들도 체크하시면 좋습니다.
파일이나 폴더를 마우스 오른쪽 클릭했을 때
VS Code로 바로 열 수 있는 메뉴가 추가됩니다.

모든 옵션을 체크했으면 다음을 클릭하세요.
```

---

## Scene 8: 설치 진행

```
설치 버튼을 클릭하면 설치가 시작됩니다.

설치는 보통 1~2분 정도 소요됩니다.
잠시만 기다려주세요.

설치가 완료되었습니다!
```

---

## Scene 9: 설치 완료 및 실행

```
"Visual Studio Code 실행" 체크박스가 선택되어 있는지 확인하고
마침 버튼을 클릭하세요.

축하합니다! VS Code가 성공적으로 설치되었습니다.
```

---

## Scene 10: 한글 언어팩 설치

```
VS Code가 실행되었습니다.
처음에는 영어로 표시되는데, 한글로 바꿔보겠습니다.

왼쪽에 보이는 블록 모양 아이콘을 클릭하세요.
이것이 확장 프로그램을 설치하는 메뉴입니다.
키보드 단축키 Ctrl + Shift + X를 눌러도 됩니다.

검색창에 Korean이라고 입력합니다.

첫 번째로 나오는 Korean Language Pack을 선택하고
Install 버튼을 클릭하세요.

설치가 완료되면 오른쪽 아래에 재시작 안내가 나타납니다.
Restart 버튼을 클릭해서 VS Code를 재시작합니다.

이제 VS Code가 한글로 표시됩니다.
메뉴와 설정이 모두 한글로 바뀐 것을 확인할 수 있습니다.
```

---

## Scene 11: 기본 인터페이스 소개

```
VS Code 기본 화면을 간단히 살펴보겠습니다.

왼쪽에 있는 아이콘들이 주요 기능입니다.
첫 번째 아이콘은 탐색기로, 파일과 폴더를 관리합니다.
두 번째는 검색 기능입니다.
세 번째는 Git 소스 관리 기능입니다.
네 번째는 프로그램을 실행하고 디버깅하는 기능입니다.
다섯 번째는 아까 사용한 확장 프로그램 설치 메뉴입니다.

상단에는 파일, 편집, 보기 등의 메뉴가 있고,
하단 상태바에는 현재 파일 정보가 표시됩니다.

이제 VS Code를 사용할 준비가 완료되었습니다!
```
