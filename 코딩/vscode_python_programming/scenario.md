```
인공지능, 데이터 분석, 웹 개발, 자동화 등

이 영상에서는 Python 설치부터

그럼 시작해볼까요?
```

## 본문 (화면녹화 + AI 나레이션)

### Scene 1: Python 설치 확인
**시간:** 약 40초

#### 화면 동작
1. 명령 프롬프트(cmd) 실행
2. `python --version` 입력
3. 설치 여부 확인

#### 나레이션
```
먼저 컴퓨터에 Python이 설치되어 있는지 확인해보겠습니다.

Windows 검색에서 cmd를 입력하고 명령 프롬프트를 실행하세요.

python --version이라고 입력하고 엔터를 누릅니다.

Python 버전이 표시되면 이미 설치되어 있는 것입니다.
"python을 찾을 수 없습니다"라고 나오면 설치가 필요합니다.
```

---

### Scene 2: Python 다운로드
**시간:** 약 1분

#### 화면 동작
1. 웹 브라우저 실행
2. 주소창에 `python.org` 입력
3. Downloads 메뉴 클릭
4. "Download Python 3.x.x" 버튼 클릭

#### 나레이션
```
Python을 설치해보겠습니다.

웹 브라우저를 열고 python.org를 입력하세요.
이곳이 Python 공식 사이트입니다.

상단 메뉴에서 Downloads를 클릭합니다.

Windows용 최신 버전 다운로드 버튼이 보입니다.
Download Python 버튼을 클릭하세요.

현재 Python 3점대 버전이 최신입니다.
다운로드가 시작됩니다.
```

---

### Scene 3: Python 설치
**시간:** 약 1분 30초

#### 화면 동작
1. 다운로드된 설치 파일 실행
2. **"Add Python to PATH" 체크** (중요!)
3. "Install Now" 클릭
4. 설치 진행 및 완료

#### 나레이션
```
다운로드가 완료되면 설치 파일을 실행합니다.

설치 화면이 나타나면 가장 중요한 옵션이 있습니다.
화면 하단의 "Add Python to PATH"를 반드시 체크하세요!

이 옵션을 체크하지 않으면 나중에 환경변수를 직접 설정해야 합니다.
꼭 체크했는지 다시 한번 확인하세요.

체크했으면 Install Now를 클릭합니다.

설치가 진행됩니다.
보통 1~2분 정도 소요됩니다.

설치가 완료되면 Close 버튼을 클릭합니다.
```

---

### Scene 4: Python 설치 확인
**시간:** 약 30초

#### 화면 동작
1. 새 명령 프롬프트 창 열기 (기존 창 닫고 새로 열기)
2. `python --version` 입력
3. 버전 정보 출력 확인

#### 나레이션
```
Python이 제대로 설치되었는지 확인해보겠습니다.

중요한 점은, 기존 명령 프롬프트를 닫고 새로 열어야 합니다.
환경변수가 적용되려면 새 창을 열어야 합니다.

새 명령 프롬프트에서 python --version을 입력합니다.

Python 버전이 표시되면 설치가 성공한 것입니다.
```

---

### Scene 5: VS Code Python 확장 설치
**시간:** 약 1분 30초

#### 화면 동작
1. VS Code 실행
2. 확장(Extensions) 아이콘 클릭 (Ctrl+Shift+X)
3. 검색창에 "Python" 입력
4. Microsoft의 "Python" 확장 선택
5. "Install" 버튼 클릭
6. 설치 완료 확인

#### 나레이션
```
이제 VS Code에서 Python을 편리하게 사용하기 위한 확장을 설치합니다.

VS Code를 실행하고, 왼쪽의 블록 모양 아이콘을 클릭하세요.
키보드 단축키 Ctrl + Shift + X를 눌러도 됩니다.

검색창에 Python을 입력합니다.

첫 번째로 나오는 Microsoft에서 만든 Python 확장을 선택하고
Install 버튼을 클릭하세요.

이 확장은 코드 자동완성, 문법 검사, 디버깅,
그리고 가상환경 관리 기능을 제공합니다.

설치가 완료될 때까지 잠시 기다려주세요.
```

---

### Scene 6: 작업 폴더 만들기
**시간:** 약 1분

#### 화면 동작
1. File > Open Folder 클릭
2. 새 폴더 만들기 (예: C:\PythonProjects)
3. 폴더 선택하여 열기
4. "Trust the authors" 선택

#### 나레이션
```
이제 VS Code에서 Python 프로젝트를 시작해보겠습니다.

파일 메뉴에서 Open Folder를 클릭하세요.
키보드 단축키 Ctrl + K, Ctrl + O를 눌러도 됩니다.

Python 프로그램을 저장할 폴더를 만들겠습니다.
원하는 위치에 PythonProjects라는 이름으로 새 폴더를 만들고 선택하세요.

폴더를 열면 보안 관련 팝업이 나타날 수 있습니다.
"Yes, I trust the authors"를 클릭하세요.

이제 왼쪽 탐색기에 빈 폴더가 표시됩니다.
```

---

### Scene 7: 첫 번째 Python 파일 만들기
**시간:** 약 1분

#### 화면 동작
1. 탐색기에서 새 파일 아이콘 클릭
2. 파일 이름 입력: `hello.py`
3. Python 인터프리터 선택 팝업 (있는 경우)

#### 나레이션
```
첫 번째 Python 파일을 만들어보겠습니다.

왼쪽 탐색기 위쪽에 있는 새 파일 아이콘을 클릭하세요.

파일 이름은 hello.py로 입력합니다.

Python 파일은 반드시 .py 확장자를 사용해야 합니다.

파일을 만들면 VS Code가 Python 인터프리터를 선택하라고 할 수 있습니다.
방금 설치한 Python 버전을 선택하세요.
```

### Scene 8: Hello World 프로그램 작성
**시간:** 약 1분 30초

#### 화면 동작
1. 코드 입력:
```python
print("Hello, World!")
```
2. 코드 자동완성 기능 보여주기
3. Ctrl+S로 저장

#### 나레이션
```
가장 기본적인 Hello World 프로그램을 작성해보겠습니다.

Python은 정말 간단합니다.
print 괄호 열고 큰따옴표 Hello, World! 큰따옴표 닫고 괄호 닫으면 끝입니다.

C언어처럼 include나 main 함수가 필요 없습니다.
세미콜론도 필요 없습니다.

이것이 Python이 배우기 쉬운 이유 중 하나입니다.

Ctrl + S를 눌러 파일을 저장합니다.
```

### Scene 9: 프로그램 실행하기 - 방법 1 (실행 버튼)
**시간:** 약 1분

#### 화면 동작
1. 오른쪽 상단의 실행 버튼(▷) 클릭
2. 하단 터미널에 결과 출력 확인

#### 나레이션
```
Python 프로그램을 실행하는 방법은 여러 가지가 있습니다.

가장 쉬운 방법은 오른쪽 상단의 실행 버튼을 클릭하는 것입니다.
재생 버튼처럼 생긴 삼각형 아이콘입니다.

버튼을 클릭하면 하단에 터미널이 열리고
프로그램이 자동으로 실행됩니다.

Hello, World!가 출력되는 것을 확인할 수 있습니다.

축하합니다! 첫 번째 Python 프로그램을 실행했습니다.
```

```
동일하게 Hello, World!가 출력됩니다.
```

### Scene 11: 다양한 출력 예제
**시간:** 약 1분 30초

#### 화면 동작
1. 코드 수정:
```python
print("Hello, World!")
print("Python을 시작합니다!")
print("VS Code로 코딩하기")
print(1 + 2 + 3)
print("결과:", 10 * 5)
```
2. 저장 후 실행
3. 결과 확인

#### 나레이션
```
print 함수를 더 활용해보겠습니다.

여러 줄을 출력하려면 print를 여러 번 사용하면 됩니다.

Python을 시작합니다와 VS Code로 코딩하기를 추가해볼게요.

print는 문자열뿐만 아니라 계산 결과도 출력할 수 있습니다.
1 + 2 + 3을 출력하면 6이 나옵니다.

문자열과 숫자를 함께 출력하려면 쉼표로 구분하면 됩니다.
"결과:" 쉼표 10 * 5 이렇게 쓰면 "결과: 50"이 출력됩니다.

저장하고 실행해보면 모든 내용이 순서대로 출력됩니다.
```

### Scene 12: 변수 사용하기
**시간:** 약 1분 30초

#### 화면 동작
1. 새 파일 만들기: `variables.py`
2. 코드 입력:
```python
name = "홍길동"
age = 25
height = 175.5

print("이름:", name)
print("나이:", age)
print("키:", height)
```
3. 실행 및 결과 확인

#### 나레이션
```
변수를 사용하는 방법도 간단히 알아보겠습니다.

새 파일 variables.py를 만듭니다.

Python에서 변수를 만들 때는 자료형을 지정할 필요가 없습니다.
name = "홍길동"이라고 쓰면 name은 문자열 변수가 됩니다.
age = 25라고 쓰면 age는 정수형 변수가 됩니다.
height = 175.5라고 쓰면 height는 실수형 변수가 됩니다.

이제 print로 변수 값을 출력해봅니다.

저장하고 실행하면 변수에 저장된 값이 출력됩니다.

Python은 이렇게 간단하게 변수를 사용할 수 있습니다.
```

### Scene 13: 사용자 입력 받기
**시간:** 약 1분 30초

#### 화면 동작
1. 새 파일 만들기: `input_example.py`
2. 코드 입력:
```python
name = input("이름을 입력하세요: ")
print("안녕하세요,", name, "님!")
```
3. 실행 및 입력 시연

#### 나레이션
```
사용자로부터 입력을 받는 방법도 알아보겠습니다.

새 파일 input_example.py를 만듭니다.

input 함수를 사용하면 사용자 입력을 받을 수 있습니다.
name = input 괄호 "이름을 입력하세요: " 괄호를 입력합니다.

그리고 입력받은 이름을 출력합니다.
print "안녕하세요," name "님!"

저장하고 실행하면 터미널에 "이름을 입력하세요:"가 표시됩니다.
이름을 입력하고 엔터를 누르면 인사 메시지가 출력됩니다.

이렇게 대화형 프로그램을 만들 수 있습니다.
```

### Scene 14: 유용한 기능 소개
**시간:** 약 1분

#### 화면 동작
1. 자동완성 시연 (pri 입력 → print 제안)
2. 오류 표시 시연 (일부러 오류 발생)
3. 코드 포맷팅 (Shift+Alt+F)

#### 나레이션
```
VS Code에서 Python 개발할 때 유용한 기능을 소개합니다.

pri만 입력해도 print가 제안됩니다.

두 번째, 오류 표시 기능입니다.
코드에 오류가 있으면 빨간 밑줄로 표시됩니다.
마우스를 올리면 오류 내용을 확인할 수 있습니다.

세 번째, 코드 포맷팅입니다.
Shift + Alt + F를 누르면 코드가 자동으로 정리됩니다.

이런 기능들을 활용하면 더 효율적으로 코딩할 수 있습니다.
```
