인트로 입니다 --------->

안녕하세요!

오늘은 VS Code에서 Python 프로그램을
작성하고 실행하는 방법을 알려드리겠습니다.

Python은 배우기 쉽고 활용 범위가 넓어서
전 세계에서 가장 인기 있는 프로그래밍 언어입니다.

인공지능, 데이터 분석, 웹 개발, 자동화 등
다양한 분야에서 사용됩니다.

이 영상에서는 Python 설치부터
첫 번째 프로그램 실행까지 전 과정을 다룹니다.

그럼 시작해볼까요?

Scene 1: Python 설치 확인 입니다 --------->

먼저 컴퓨터에 Python이 설치되어 있는지 확인해보겠습니다.

Windows 검색에서 cmd를 입력하고 명령 프롬프트를 실행하세요.

python --version이라고 입력하고 엔터를 누릅니다.

Python 버전이 표시되면 이미 설치되어 있는 것입니다.
"python을 찾을 수 없습니다"라고 나오면 설치가 필요합니다.

Scene 2: Python 다운로드 입니다 --------->

Python을 설치해보겠습니다.

웹 브라우저를 열고 python.org를 입력하세요.
이곳이 Python 공식 사이트입니다.

상단 메뉴에서 Downloads를 클릭합니다.

Windows용 최신 버전 다운로드 버튼이 보입니다.
Download Python 버튼을 클릭하세요.

현재 Python 3점대 버전이 최신입니다.
다운로드가 시작됩니다.

Scene 3: Python 설치 입니다 --------->

다운로드가 완료되면 설치 파일을 실행합니다.

설치 화면이 나타나면 가장 중요한 옵션이 있습니다.
화면 하단의 "Add Python to PATH"를 반드시 체크하세요!

이 옵션을 체크하지 않으면 나중에 환경변수를 직접 설정해야 합니다.
꼭 체크했는지 다시 한번 확인하세요.

체크했으면 Install Now를 클릭합니다.

설치가 진행됩니다.
보통 1~2분 정도 소요됩니다.

설치가 완료되면 Close 버튼을 클릭합니다.

Scene 4: Python 설치 확인 입니다 --------->

Python이 제대로 설치되었는지 확인해보겠습니다.

중요한 점은, 기존 명령 프롬프트를 닫고 새로 열어야 합니다.
환경변수가 적용되려면 새 창을 열어야 합니다.

새 명령 프롬프트에서 python --version을 입력합니다.

Python 버전이 표시되면 설치가 성공한 것입니다.

Scene 5: VS Code Python 확장 설치 입니다 --------->

이제 VS Code에서 Python을 편리하게 사용하기 위한 확장을 설치합니다.

VS Code를 실행하고, 왼쪽의 블록 모양 아이콘을 클릭하세요.
키보드 단축키 Ctrl + Shift + X를 눌러도 됩니다.

검색창에 Python을 입력합니다.

첫 번째로 나오는 Microsoft에서 만든 Python 확장을 선택하고
Install 버튼을 클릭하세요.

이 확장은 코드 자동완성, 문법 검사, 디버깅,
그리고 가상환경 관리 기능을 제공합니다.

설치가 완료될 때까지 잠시 기다려주세요.

Scene 6: 작업 폴더 만들기 입니다 --------->

이제 VS Code에서 Python 프로젝트를 시작해보겠습니다.

파일 메뉴에서 Open Folder를 클릭하세요.
키보드 단축키 Ctrl + K, Ctrl + O를 눌러도 됩니다.

Python 프로그램을 저장할 폴더를 만들겠습니다.
원하는 위치에 PythonProjects라는 이름으로 새 폴더를 만들고 선택하세요.

폴더를 열면 보안 관련 팝업이 나타날 수 있습니다.
"Yes, I trust the authors"를 클릭하세요.

이제 왼쪽 탐색기에 빈 폴더가 표시됩니다.

Scene 7: 첫 번째 Python 파일 만들기 입니다 --------->

첫 번째 Python 파일을 만들어보겠습니다.

왼쪽 탐색기 위쪽에 있는 새 파일 아이콘을 클릭하세요.

파일 이름은 hello.py로 입력합니다.

Python 파일은 반드시 .py 확장자를 사용해야 합니다.

파일을 만들면 VS Code가 Python 인터프리터를 선택하라고 할 수 있습니다.
방금 설치한 Python 버전을 선택하세요.

Scene 8: Hello World 프로그램 작성 입니다 --------->

가장 기본적인 Hello World 프로그램을 작성해보겠습니다.

Python은 정말 간단합니다.

AI를 이용하여 python 코드를 만들어 보겠읍니다.

chatGPT를 열고, prompt에 'Hello, world를 print하는 python program을 만들어 주세요.'
를 입력하고, 실행을 누릅니다.

만들어진 code를 copy하여 vscode의 file에 붙여넣기 합니다.

Ctrl + S를 눌러 파일을 저장합니다.


Scene 9: 프로그램 실행하기 - 방법 1 (실행 버튼) 입니다 --------->

Python 프로그램을 실행하는 방법은 여러 가지가 있습니다.

가장 쉬운 방법은 오른쪽 상단의 실행 버튼을 클릭하는 것입니다.
재생 버튼처럼 생긴 삼각형 아이콘입니다.

버튼을 클릭하면 하단에 터미널이 열리고
프로그램이 자동으로 실행됩니다.

Hello, World!가 출력되는 것을 확인할 수 있습니다.

축하합니다! 첫 번째 Python 프로그램을 실행했습니다.

Scene 10: 프로그램 실행하기 - 방법 2 (터미널) 입니다 --------->

터미널에서 직접 실행하는 방법도 알아보겠습니다.

상단 메뉴에서 Terminal, New Terminal을 클릭하세요.
키보드 단축키 Ctrl + 백틱을 눌러도 됩니다.

터미널에 python hello.py라고 입력하고 엔터를 누릅니다.

동일하게 Hello, World!가 출력됩니다.

이 방법은 명령줄 인자를 전달하거나
여러 파일을 실행할 때 유용합니다.


Scene 14: 유용한 기능 소개 입니다 --------->

VS Code에서 Python 개발할 때 유용한 기능을 소개합니다.

첫 번째, 자동완성 기능입니다.
pri만 입력해도 print가 제안됩니다.
탭이나 엔터를 누르면 자동으로 완성됩니다.

두 번째, 오류 표시 기능입니다.
코드에 오류가 있으면 빨간 밑줄로 표시됩니다.
마우스를 올리면 오류 내용을 확인할 수 있습니다.

세 번째, 코드 포맷팅입니다.
Shift + Alt + F를 누르면 코드가 자동으로 정리됩니다.

이런 기능들을 활용하면 더 효율적으로 코딩할 수 있습니다.

아웃트로 입니다 --------->

오늘은 VS Code에서 Python 프로그램을
작성하고 실행하는 방법을 알아보았습니다.

정리하면,
첫째, Python 설치 시 PATH 추가 체크
둘째, VS Code Python 확장 설치
셋째, .py 파일 만들고 코드 작성
넷째, 실행 버튼 또는 터미널로 실행
이 네 단계를 기억하시면 됩니다.

좋아요와 구독 부탁드립니다.
