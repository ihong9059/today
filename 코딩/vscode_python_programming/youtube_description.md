❓ 자주 묻는 질문

Q. "python을 찾을 수 없습니다" 오류가 나요
A. Python 설치 시 "Add Python to PATH"를 체크하지 않은 경우입니다.
   Python을 재설치하거나 환경변수를 수동으로 추가하세요.

Q. 한글이 깨져요
A. 파일 인코딩을 UTF-8로 설정하세요.
   VS Code 하단 상태바에서 인코딩을 확인할 수 있습니다.

Q. 실행 버튼이 안 보여요
A. VS Code Python 확장이 설치되어 있는지 확인하세요.
