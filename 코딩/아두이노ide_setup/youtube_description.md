## 설명 (Description)

```
아두이노 IDE 설치 방법을 처음부터 끝까지 쉽게 알려드립니다!
LED 제어부터 로봇, IoT 프로젝트까지 - 아두이노의 첫 걸음을 함께 시작해보세요.

📌 이 영상에서 배우는 내용
• 아두이노 공식 사이트에서 IDE 다운로드하기
• 아두이노 IDE 2.x 버전 설치하기
• USB 드라이버 설치하기
• 보드 매니저 & 라이브러리 매니저 사용법
• 기본 인터페이스 살펴보기

⏱️ 타임라인

4:30 첫 실행 및 초기 설정

9:00 마무리

🔗 아두이노 공식 다운로드
https://www.arduino.cc/en/software

💡 설치 시 꼭 알아둘 것
✅ Windows 10 이상: 64bit MSI 파일 선택
✅ USB 드라이버 설치 창이 나오면 반드시 "설치" 클릭
✅ 첫 실행 시 추가 구성요소 다운로드 대기

🛠️ 아두이노란?
• 전 세계에서 가장 인기 있는 오픈소스 하드웨어 플랫폼
• LED 제어, 센서, 모터, 로봇, IoT 등 다양한 프로젝트 가능
• 아두이노 우노(Uno), 나노(Nano), 메가(Mega) 등 다양한 보드 지원

📦 지원 보드
• Arduino Uno / Nano / Mega (기본 설치)
• ESP32 / ESP8266 (보드 매니저에서 추가 설치)
• 기타 호환 보드

📚 유용한 라이브러리
• LiquidCrystal - LCD 디스플레이
• DHT - 온습도 센서
• Servo - 서보 모터
• Wire - I2C 통신
```
