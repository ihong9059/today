# 아두이노 IDE 설치 동영상 시나리오

**영상 제목:** 아두이노 IDE 설치 방법 | Windows 완벽 가이드
**예상 길이:** 약 8~10분
**대상:** 아두이노 입문자, 메이커, DIY 전자공작 초보자

---

## 인트로 (Vrew 텍스트→비디오)

**시간:** 약 40초

### 나레이션 대본

```
안녕하세요!

오늘은 아두이노 프로그래밍의 첫 걸음,
아두이노 IDE를 설치하는 방법을 알려드리겠습니다.

아두이노는 전 세계에서 가장 인기 있는 오픈소스 하드웨어 플랫폼으로,
LED 제어부터 로봇, IoT 프로젝트까지 다양한 것을 만들 수 있습니다.

이 영상을 따라하시면 10분 안에
아두이노 IDE 설치를 완료하실 수 있습니다.

그럼 지금 바로 시작해볼까요?
```

### 화면 구성 (AI 생성)
- 아두이노 보드, 전자부품 관련 배경 이미지
- 아두이노 로고 등장
- 텍스트 애니메이션: "아두이노 IDE 설치 가이드"

---

## 본문 (화면녹화 + AI 나레이션)

### Scene 1: 아두이노 공식 사이트 접속
**시간:** 약 40초

#### 화면 동작
1. 웹 브라우저(Chrome/Edge) 실행
2. 주소창에 `arduino.cc` 입력
3. 공식 사이트 메인 페이지 표시
4. 상단 메뉴에서 "Software" 클릭

#### 나레이션
```
먼저 웹 브라우저를 실행합니다.

주소창에 arduino.cc를 입력하고 엔터를 누르세요.
이곳이 아두이노 공식 사이트입니다.

상단 메뉴에서 Software를 클릭합니다.
여기서 아두이노 IDE를 다운로드할 수 있습니다.
```

---

### Scene 2: 다운로드 페이지
**시간:** 약 50초

#### 화면 동작
1. Software 페이지에서 "Arduino IDE" 섹션 확인
2. "Windows" 옵션 선택 (Windows 10 and newer, 64 bits)
3. 다운로드 옵션 설명

#### 나레이션
```
다운로드 페이지가 나타났습니다.

아두이노 IDE 2점대 버전이 최신 버전입니다.
화면에서 Windows용 다운로드 옵션을 찾아주세요.

Windows 10 이상을 사용하신다면
"Windows 10 and newer, 64 bits"를 선택하시면 됩니다.

참고로 ZIP 파일과 MSI 설치 파일 두 가지가 있는데,
MSI 파일로 설치하는 것이 더 간편합니다.
```

---

### Scene 3: 기부 페이지 및 다운로드
**시간:** 약 30초

#### 화면 동작
1. 다운로드 버튼 클릭
2. 기부 페이지 표시
3. "Just Download" 버튼 클릭
4. 다운로드 시작 확인

#### 나레이션
```
다운로드 버튼을 클릭하면 기부 페이지가 나타납니다.

아두이노 프로젝트를 후원하고 싶으시면 기부하셔도 좋고,
무료로 다운로드하려면 "Just Download" 버튼을 클릭하세요.

다운로드가 시작됩니다.
파일 크기는 약 150메가바이트 정도입니다.
```

---

### Scene 4: 설치 파일 실행
**시간:** 약 30초

#### 화면 동작
1. 다운로드된 `arduino-ide_2.x.x_Windows_64bit.exe` 파일 클릭
2. 사용자 계정 컨트롤(UAC) 창에서 "예" 클릭
3. 설치 마법사 시작 화면 표시

#### 나레이션
```
다운로드가 완료되면 설치 파일을 실행합니다.

윈도우에서 설치를 허용할지 물어보면 "예"를 클릭하세요.

잠시 후 아두이노 IDE 설치 마법사가 나타납니다.
```

---

### Scene 5: 라이선스 동의
**시간:** 약 20초

#### 화면 동작
1. 라이선스 계약서 화면
2. "I Agree" 버튼 클릭

#### 나레이션
```
라이선스 계약서 화면입니다.
내용을 확인하시고 "I Agree" 버튼을 클릭하세요.
```

---

### Scene 6: 설치 옵션 선택
**시간:** 약 30초

#### 화면 동작
1. 설치 옵션 화면 (Install for all users / current user)
2. 적절한 옵션 선택
3. "Next" 버튼 클릭

#### 나레이션
```
설치 옵션을 선택하는 화면입니다.

모든 사용자가 사용하려면 "Install for all users"를,
현재 사용자만 사용하려면 "Install for current user only"를 선택하세요.

일반적으로 기본 옵션 그대로 진행하시면 됩니다.
Next를 클릭합니다.
```

---

### Scene 7: 설치 위치 선택
**시간:** 약 20초

#### 화면 동작
1. 설치 경로 선택 화면
2. 기본 경로 유지
3. "Install" 버튼 클릭

#### 나레이션
```
설치 위치를 선택하는 화면입니다.
특별한 이유가 없다면 기본 경로를 그대로 사용하세요.

Install 버튼을 클릭하면 설치가 시작됩니다.
```

---

### Scene 8: 설치 진행
**시간:** 약 40초

#### 화면 동작
1. 설치 진행 상태 표시 (프로그레스 바)
2. 드라이버 설치 팝업 (있는 경우 "설치" 클릭)
3. 설치 완료 화면

#### 나레이션
```
설치가 진행됩니다.
보통 2~3분 정도 소요됩니다.

설치 중에 USB 드라이버 설치 창이 나타날 수 있습니다.
아두이노 보드를 컴퓨터와 연결하기 위해 필요하니
"설치" 버튼을 클릭해주세요.

설치가 완료되었습니다!
```

---

### Scene 9: 설치 완료 및 실행
**시간:** 약 20초

#### 화면 동작
1. "Run Arduino IDE" 체크박스 확인
2. "Finish" 버튼 클릭
3. 아두이노 IDE 첫 실행 화면

#### 나레이션
```
"Run Arduino IDE" 체크박스가 선택되어 있는지 확인하고
Finish 버튼을 클릭하세요.

축하합니다! 아두이노 IDE가 성공적으로 설치되었습니다.
```

---

### Scene 10: 첫 실행 및 초기 설정
**시간:** 약 1분

#### 화면 동작
1. 아두이노 IDE 첫 실행 화면
2. 추가 구성요소 다운로드 진행 (있는 경우)
3. 기본 인터페이스 표시

#### 나레이션
```
아두이노 IDE가 처음 실행되면
필요한 추가 구성요소를 자동으로 다운로드합니다.

인터넷 연결 상태에 따라 몇 분 정도 걸릴 수 있습니다.
잠시만 기다려주세요.

초기 설정이 완료되면 아두이노 IDE 메인 화면이 나타납니다.
```

---

### Scene 11: 보드 매니저 확인
**시간:** 약 1분 30초

#### 화면 동작
1. 왼쪽 사이드바에서 보드 매니저 아이콘 클릭
2. "Arduino AVR Boards" 확인 (기본 설치됨)
3. 필요시 추가 보드 검색 및 설치 방법 안내

#### 나레이션
```
아두이노 IDE의 주요 기능을 살펴보겠습니다.

왼쪽에 있는 보드 모양 아이콘을 클릭하세요.
이것이 보드 매니저입니다.

여기서 다양한 아두이노 보드를 설치하고 관리할 수 있습니다.
기본적으로 Arduino AVR Boards가 설치되어 있어서
아두이노 우노, 나노, 메가 등을 바로 사용할 수 있습니다.

다른 보드, 예를 들어 ESP32나 ESP8266을 사용하려면
여기서 해당 보드를 검색해서 설치하면 됩니다.
```

---

### Scene 12: 라이브러리 매니저 확인
**시간:** 약 1분

#### 화면 동작
1. 왼쪽 사이드바에서 라이브러리 매니저 아이콘 클릭
2. 라이브러리 검색 및 설치 방법 간단 안내

#### 나레이션
```
그 아래 책 모양 아이콘은 라이브러리 매니저입니다.

센서나 디스플레이 같은 부품을 사용할 때
필요한 라이브러리를 여기서 검색하고 설치할 수 있습니다.

예를 들어 LCD를 사용하려면 "LiquidCrystal"을 검색하고,
온습도 센서 DHT를 사용하려면 "DHT"를 검색하면 됩니다.
```

---

### Scene 13: 기본 인터페이스 소개
**시간:** 약 1분

#### 화면 동작
1. 상단 툴바 설명 (확인, 업로드, 디버그, 보드 선택)
2. 코드 편집 영역
3. 하단 출력 창
4. 시리얼 모니터 버튼

#### 나레이션
```
아두이노 IDE 기본 화면을 살펴보겠습니다.

상단에 있는 체크 표시 버튼은 코드를 검증하는 버튼입니다.
화살표 버튼은 아두이노 보드에 코드를 업로드하는 버튼입니다.

가운데 흰색 영역이 코드를 작성하는 곳입니다.
기본적으로 setup과 loop 함수가 있는
템플릿 코드가 표시되어 있습니다.

하단에는 컴파일 결과와 오류 메시지가 표시됩니다.

오른쪽 위의 돋보기 아이콘은 시리얼 모니터입니다.
아두이노와 컴퓨터 간의 통신 내용을 확인할 때 사용합니다.
```
