## 1. 개요

### 법적 근거
- **전기안전관리법 제22조 제1항**: 자가용 전기설비의 소유자 또는 점유자는 「국가기술자격법」에 따른 **전기·기계·토목 분야**의 기술자격을 취득한 사람 중에서 전기안전관리자를 선임해야 함
- **전기사업법 제73조 제1항**: 전기사업자는 일정한 자격을 갖춘 안전관리자를 전기설비 사용 전 검사신청 전 또는 사업개시 전에 선임해야 함

### 중요 참고사항
> **주의:** 신재생에너지발전설비기사(태양광) 자격증만으로는 전기안전관리자로 선임될 수 없습니다. 이 자격증은 "시공" 분야에만 활용 가능하며, 안전관리 업무를 위해서는 **전기기사** 또는 **전기산업기사** 등 전기 분야 자격이 필요합니다.

## 2. 태양광 발전소 전기안전관리자 선임 기준

### 선임 의무 대상
| 구분 | 용량 기준 |
|------|-----------|
| 선임 대상 | 태양광 설비 **20kW 초과** |
| 계약전력 기준 | **75kW 이상** 사업장 |
| 특고압(22.9kV 초과) | 용량 무관, **무조건 선임** |

### 상주 선임 vs 대행 가능 범위

| 구분 | 용량 기준 | 비고 |
|------|-----------|------|
| **상주 선임 필수** | 3,000kW 이상 | 원격감시 미설치 시 1,000kW 이상 |
| **대행 가능 (법인)** | 3MW 미만 | 원격감시제어시스템 필요 |
| **대행 가능 (개인)** | 750kW 이하 | 원격감시제어시스템 필요 |

### 미선임 시 처벌
- **500만원 이하의 벌금**

## 3. 전기안전관리자 자격 종류

### 3.1 전기기술사
| 항목 | 내용 |
|------|------|
| **선임 범위** | 모든 전기설비 (무제한) |
| **경력 요건** | 없음 |
| **취득 난이도** | 최상 |

### 3.2 전기기사 / 전기기능장
| 항목 | 내용 |
|------|------|
| **응시자격** | 4년제 관련학과 졸업, 산업기사 + 1년 경력, 기능사 + 3년 경력 등 |
| **시험과목** | 전기자기학, 전력공학, 전기기기, 회로이론, 제어공학, 전기설비기술기준 |
| **합격기준** | 필기/실기 각 60점 이상 |

**선임 기준:**
| 설비 조건 | 필요 경력 |
|-----------|-----------|
| 1,500kW 미만 | 경력 불필요 |
| 2,000kW 미만, 10만V 미만 | 1년 이상 |
| 모든 설비 (무제한 선임) | **2년 이상** |

### 3.3 전기산업기사
| 항목 | 내용 |
|------|------|
| **응시자격** | 2~3년제 관련학과 졸업, 기능사 + 1년 경력, 동일분야 2년 경력 등 |
| **시험과목** | 전기자기학, 전력공학, 전기기기, 회로이론, 전기설비기술기준 |
| **합격기준** | 필기/실기 각 60점 이상 |

**선임 기준:**
| 설비 조건 | 필요 경력 |
|-----------|-----------|
| 1,500kW 미만, 10만V 미만 | 경력 불필요 |
| 2,000kW 미만, 10만V 미만 | 2년 이상 |
| 10만V 미만 전체 | **4년 이상** |

> **참고:** 전기산업기사는 경력 없이는 선임 자격이 제한적이며, 경력을 쌓아야 자격증 선임이 가능

### 3.4 전기기능사
| 항목 | 내용 |
|------|------|
| **응시자격** | 제한 없음 (학력, 경력, 나이 무관) |
| **시험횟수** | 연 4회 |
| **시험과목(필기)** | 전기이론(직류/교류회로), 정전계, 정자계 (60문제, 60분) |
| **시험과목(실기)** | 전기설비 작업 (4시간 30분) |
| **합격기준** | 필기/실기 각 60점 이상 (필기 과락 없음) |
| **합격률** | 약 60% |
| **선임 범위** | 제한적 (보조원 역할) |
| **활용** | 경력 축적 후 산업기사/기사 응시 |

> **참고:** 전기기능사는 학력/경력이 없는 분들이 전기 분야에 입문할 때 취득하는 자격증입니다. 4년제 공학 관련학과 졸업자는 전기기능사 없이 바로 전기기사 응시가 가능합니다.

## 4. 신재생에너지 발전설비 자격증

> **주의:** 아래 자격증들은 **시공** 분야에만 활용 가능하며, **전기안전관리자 선임에는 사용할 수 없습니다.**

### 4.1 신재생에너지발전설비기사(태양광)

| 항목 | 내용 |
|------|------|
| **시행연도** | 2013년~ |
| **시행기관** | 한국산업인력공단 |
| **응시자격** | 관련학과 졸업자, 관련분야 2년 이상 경력자 |
| **시험과목(필기)** | 태양광발전시스템설계, 태양광발전시스템시공, 태양광발전시스템운용 및 유지보수, 신재생에너지관련법규 |
| **시험과목(실기)** | 태양광발전시스템 설계 및 시공 실무 |
| **활용 분야** | 태양광 발전 시스템 시공 |

### 4.2 신재생에너지발전설비산업기사(태양광)

| 항목 | 내용 |
|------|------|
| **응시자격** | 학점은행제 41학점 이수, 동일분야 2년 경력, 기능사 + 1년 경력 |
| **특별 경로** | 전기공사인재개발원 600시간 과정 이수 (국비 지원) |
| **2025년 시험** | 연 3회 실시 |
| **평균 연봉** | 약 5,393만원 (전체 평균 4,072만원 대비 높음) |

### 4.3 신재생에너지발전설비기능사(태양광)

| 항목 | 내용 |
|------|------|
| **시행연도** | 2013년 1월~ |
| **응시자격** | 제한 없음 |
| **응시료** | 필기 14,500원 / 실기 17,200원 |
| **2025년 시험** | 2회, 3회, 4회 실시 |

## 5. 설비 규모별 필요 자격

### 태양광 발전소 전기안전관리자 자격 요약

| 설비 용량 | 필요 자격 |
|-----------|-----------|
| **20kW 이하** | 선임 불필요 |
| **20kW 초과 ~ 750kW (개인)** | 대행 가능 (원격감시 설치 시) |
| **20kW 초과 ~ 1,500kW** | 전기산업기사 (경력 불필요) |
| **1,500kW ~ 2,000kW, 10만V 미만** | 전기기사 + 1년 경력 또는 전기산업기사 + 2년 경력 |
| **2,000kW 이상** | 전기기사 + 2년 경력 또는 전기산업기사 + 4년 경력 |
| **3,000kW 이상** | 상주 선임 필수 |

## 6. 자격증 취득 경로

### 추천 경로 1: 4년제 공학 관련학과 졸업자 (가장 빠른 경로)

```
4년제 공학 관련학과 졸업 (전기, 전자, 기계 등)
    ↓ (전기기능사 불필요!)
전기기사 바로 응시 → 취득
    ↓ 2년 경력
전기안전관리자 무제한 선임 자격
```

**해당 학과 예시:**
- 전기공학과, 전자공학과, 전기전자공학과
- 제어계측공학과, 메카트로닉스공학과
- 기타 전기·전자 관련 공학과

> **예시:** 아주대학교 전자공학과 졸업자 → 전기기사 바로 응시 가능

### 추천 경로 2: 전문대(2~3년제) 졸업자

```
전기 관련 전문대 졸업
    ↓
전기산업기사 (바로 응시)
    ↓ 경력 축적
전기기사 응시 또는 선임 자격 확보
```

### 추천 경로 3: 비전공자 - 전기기능사부터 시작

```
전기기능사 (응시자격 없음)
    ↓ 1년 경력
전기산업기사
    ↓ 1년 경력 또는 학점은행제
전기기사
    ↓ 2년 경력
무제한 선임 자격
```

### 추천 경로 4: 학점은행제 활용 (비전공자)

```
학점은행제 41학점 이수 (온라인)
    ↓
전기산업기사 응시 자격 획득
    ↓
자격증 취득 후 경력 축적
```

### 학력별 응시 가능 자격증 요약

| 학력 | 전기기능사 | 전기산업기사 | 전기기사 |
|------|-----------|-------------|---------|
| 학력 무관 | O | X | X |
| 전문대 관련학과 졸업 | O | **O (바로 응시)** | X |
| 4년제 관련학과 졸업 | O | O | **O (바로 응시)** |
| 기능사 + 1년 경력 | - | O | X |
| 산업기사 + 1년 경력 | - | - | O |

## 7. 취업 전망 및 연봉

### 시장 전망
- 신재생에너지 관련 산업 고용 인원: 2010년 1.2만명 → 2020년 8.1만명 (연평균 21.2% 성장)
- 태양광 산업: 2010년 2천명 → 2020년 2.9만명 급증

### 평균 연봉
| 직종 | 평균 연봉 (중앙값) |
|------|-------------------|
| 신재생에너지발전설비산업기사(태양광) 관련 | **5,393만원** |
| 전체 직업 평균 | 4,072만원 |

### 규제 완화 동향 (2025년 전망)
- 전기기사 실무경력: 2년 → **1년 6개월**로 완화 예정
- 전기산업기사 실무경력: 4년 → **3년**으로 완화 예정

## 요약

### 태양광 발전소 전기안전관리자가 되려면?

1. **필수 자격증**: 전기기사 또는 전기산업기사 (신재생에너지발전설비기사는 안전관리자 불가)
2. **최소 요건**: 전기산업기사 취득 시 1,500kW 미만 설비 선임 가능 (경력 불필요)
3. **무제한 선임**: 전기기사 + 2년 경력 또는 전기산업기사 + 4년 경력
4. **소규모 설비**: 750kW 이하(개인) 또는 3MW 미만(법인)은 대행 가능
