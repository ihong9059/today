### 1.2 주요 제품군

#### SHREDDER (파쇄기)
- 이축(二軸) 회전 방식, 고토크(High Torque) 파쇄
- 도시형 대형 쓰레기 ~ 산업폐기물 처리
- 저진동, 저소음, 저공해 설계
- 대량 파쇄 가능

#### CRUSHER & MILL (분쇄기)
- 다양한 입도의 분쇄 처리
- 산업별 맞춤형 분쇄 솔루션

#### PIN MILL
- 고속 충격식 분쇄
- 미분 처리 특화

## 4. 구체적 프로젝트 제안

### 프로젝트 A: 스마트 파쇄기 제어 시스템 개발

#### 개요
| 항목 | 내용 |
|------|------|
| **프로젝트명** | Smart Shredder Control System (SSCS) |
| **목표** | 파쇄기 지능형 제어 및 효율 최적화 |
| **담당** | 임호균 (H/W, F/W), 홍광선 (IoT 연동) |

#### 주요 기능
1. **적응형 토크 제어**
   - 파쇄물 종류/양에 따른 자동 토크 조절
   - 과부하 보호 및 자동 역회전

2. **실시간 모니터링**
   - 모터 전류, 온도, 진동 실시간 감시
   - 이상 징후 즉시 알림

3. **에너지 효율 최적화**
   - 부하에 따른 최적 속도 제어
   - 대기 전력 최소화

#### 개발 범위

| 구분 | 내용 | 담당 |
|------|------|------|
| **메인 제어보드** | ARM Cortex-M4 기반 MCU 보드 | 임호균 |
| **모터 드라이버** | BLDC/인버터 제어 회로 | 임호균 |
| **센서 인터페이스** | 진동/온도/전류 센서 연동 | 임호균 |
| **통신 모듈** | Modbus RTU/TCP, RS485 | 임호균 |
| **IoT Gateway** | LoRaWAN/Zigbee 연동 | 홍광선 |
| **원격 모니터링** | 웹 대시보드 연동 | 홍광선 |

#### 예상 산출물
- 메인 제어보드 (PCB + Firmware)
- 센서 인터페이스 보드
- 통신 프로토콜 규격서
- 사용자 매뉴얼
