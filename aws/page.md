# page 구조
## 1 이 page에서 구현할 목표와 목적 설명
## 2 동영상(전체 과정 설명 동영상)
## 3 ai에 질문할 prompt 창
### 3.1 적용할 무료 ai의 url button
## 4 ai로 부터 나온 결과 display 또는 download
## 5 실행 결과 및 다음질문 진행 button

설명:
1에서는 이 page에서 진행할 교육 목표와 목적에 대해서 설명하고,
2에서는 사전에 이 page에서 진행할 내용을 동영상으로 만들어 놓은것을 display하도록 하고
3에서는 동영상에 나온것과 같은 방법으로 ai에 질문할 내용을 입력하고,
3.1에서는 여러 무료 ai의 url에 button을 매칭시켜놓아서, 해당 button을 누르면 해당 ai가 나타나고, 거기에 질문내용이 자동 입력되고
4에서는 ai로 부터 나온 결과가, 교육자가 원하는 결과인지를 확인하고, 아니면 3으로 돌아가서 재 질문을 하던지, ai를 바꾸는 작업을 진행해서
원하는 답변이 나오도록 유도.
5에서는 이번 session이 끝나면, 다음 session으로 옮겨서 해당 page의 목표가 달성되면 다음 page로 이동하도록 한다.
