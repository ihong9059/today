## 1. 프로젝트 이해

### 1.1 프로젝트 개요
침대 브랜드의 온라인 마케팅 캠페인으로, 사용자가 꿈 내용을 입력하면 **AI가 해몽**해주는 인터랙티브 이벤트 페이지를 제작하는 프로젝트입니다.

### 1.2 핵심 목표

| 목표 | 내용 |
|------|------|
| **브랜드 경험** | 웰메이드 디자인 + 엔터테인먼트 요소로 즐거운 경험 제공 |
| **바이럴 유도** | SNS 공유 최적화로 자발적 확산 유도 |
| **모바일 최적화** | 모바일 우선 설계, 몰입감 있는 인터페이스 |
| **AI 활용** | GPT 기반 꿈 해몽 + 브랜드 제품 매칭 |

### 1.3 핵심 평가 요소 인식


## 4. 상세 기능 구현 계획

### 4.1 인트로 & 로딩 애니메이션

#### 인트로 시퀀스
```
[0s]     브랜드 로고 페이드인
         │
[0.5s]   로고 → 달/별 모티브로 변형 (MorphSVG)
         │
[1.5s]   별똥별 파티클 효과 + 배경 확장
         │
[2.5s]   "당신의 꿈을 들려주세요" 타이핑
         │
[3.5s]   CTA 버튼 등장 + 메인 화면 전환
```

#### 코드 예시 (GSAP)
```javascript
// 인트로 타임라인
const introTimeline = gsap.timeline();

introTimeline
  .from('.logo', {

    duration: 1,
    ease: 'power2.inOut'
  })
  .from('.stars', {
    opacity: 0,
    stagger: 0.05,

  }, '-=0.5')
  .from('.title', {
    opacity: 0,

    duration: 0.8
  })
  .from('.cta-button', {
    opacity: 0,
    scale: 0.8,
    duration: 0.5,
    ease: 'elastic.out'
  });
```

```jsx
// 꿈 입력 컴포넌트
const DreamInput = () => {
  const [dream, setDream] = useState('');
  const inputRef = useRef(null);

  // 입력 시 파티클 효과
  const handleInput = (e) => {
    setDream(e.target.value);

    // 글자 입력마다 반짝임
    gsap.to('.sparkle', {
      opacity: 1,
      scale: 1.2,
      duration: 0.2,
      yoyo: true,
      repeat: 1,
    });
  };

  return (
    <motion.div
      initial={{ opacity: 0, y: 50 }}
      animate={{ opacity: 1, y: 0 }}
      transition={{ duration: 0.8, ease: 'easeOut' }}
    >
      <textarea
        ref={inputRef}
        value={dream}
        onChange={handleInput}
        placeholder="어젯밤 꾼 꿈을 자유롭게 적어주세요..."
        maxLength={200}
        className="dream-textarea"
      />
      <div className="char-count">{dream.length}/200</div>
    </motion.div>
  );
};
```

### 4.3 AI 분석 로딩 시퀀스

#### 로딩 애니메이션 컨셉
```
Phase 1: "꿈의 파편을 수집하고 있어요..."
         [별들이 모이는 애니메이션]
              ↓
Phase 2: "꿈의 의미를 해석하고 있어요..."
         [책이 펼쳐지며 글자가 나타나는 효과]
              ↓
Phase 3: "당신만의 해몽을 준비하고 있어요..."
         [수정구슬에 이미지가 맺히는 효과]
              ↓
Phase 4: 결과 화면으로 전환
         [문이 열리는 듯한 전환 효과]
```

#### Lottie 애니메이션 활용
```jsx
// 로딩 시퀀스 컴포넌트
const LoadingSequence = ({ phase }) => {
  const phases = [
    { text: '꿈의 파편을 수집하고 있어요...', lottie: 'stars-gathering' },
    { text: '꿈의 의미를 해석하고 있어요...', lottie: 'book-opening' },
    { text: '당신만의 해몽을 준비하고 있어요...', lottie: 'crystal-ball' },
  ];

  return (
    <motion.div
      key={phase}
      initial={{ opacity: 0 }}
      animate={{ opacity: 1 }}
      exit={{ opacity: 0 }}
      className="loading-container"
    >
      <Lottie
        animationData={phases[phase].lottie}
        loop={true}
        className="loading-animation"
      />
      <motion.p
        initial={{ opacity: 0, y: 20 }}
        animate={{ opacity: 1, y: 0 }}
        className="loading-text"
      >
        {phases[phase].text}
      </motion.p>
    </motion.div>
  );
};
```

### 4.5 AI 프롬프트 엔지니어링

#### GPT-4o 프롬프트 설계
```javascript
const generateDreamInterpretation = async (dreamContent, userName) => {
  const systemPrompt = `
당신은 따뜻하고 긍정적인 꿈 해몽 전문가입니다.
[브랜드명] 침대 브랜드의 마케팅 캠페인을 위해 꿈을 해석합니다.

## 역할
- 사용자의 꿈을 분석하여 긍정적이고 희망적인 해석을 제공
- 브랜드 톤앤매너: 따뜻함, 편안함, 꿈의 소중함
- 자연스럽게 '좋은 잠'과 '꿈'의 연결성 강조

## 응답 형식 (JSON)
{
  "title": "꿈의 제목 (5자 이내)",
  "interpretation": "해몽 내용 (150자 이내, 긍정적 톤)",
  "luckyItem": "행운의 침구 아이템 (베개/이불/매트리스 중 택1)",
  "luckyMessage": "행운 아이템과 연결된 메시지 (30자 이내)",
  "emoji": "꿈과 어울리는 이모지 1개"
}

## 주의사항
- 부정적이거나 불길한 해석 금지
- 모든 꿈을 긍정적으로 재해석
- 악몽도 "변화의 시작", "성장의 기회"로 해석
`;

  const userPrompt = `
사용자 이름: ${userName}
꿈 내용: ${dreamContent}

위 꿈을 해몽해주세요.
`;

  const response = await openai.chat.completions.create({
    model: 'gpt-4o',
    messages: [
      { role: 'system', content: systemPrompt },
      { role: 'user', content: userPrompt },
    ],
    temperature: 0.7,
    max_tokens: 500,
    response_format: { type: 'json_object' },
  });

  return JSON.parse(response.choices[0].message.content);
};
```
