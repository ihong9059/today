# BLE Mesh Bicycle Parking Lock System
## 일본 자전거 주차장용 BLE Mesh 기반 잠금 제어 시스템

---

## 1. 시스템 개요

### 1.1 제품 소개

BLE Mesh 무선 통신 기술을 활용한 **스마트 자전거 주차장 잠금 제어 시스템**입니다.
중앙 정산기와 각 주차 슬롯의 잠금장치가 BLE Mesh 네트워크로 연결되어,
자전거의 입/출차 및 요금 정산을 자동화합니다.

### 1.2 적용 분야
- 일본 공공 자전거 주차장
- 상업시설 자전거 보관소
- 역/터미널 주변 자전거 주차장
- 아파트/오피스 자전거 보관소

### 1.3 핵심 가치

| 가치 | 설명 |
|------|------|
| **무인 운영** | 24시간 무인 자동 운영 |
| **편의성** | 간편한 정산 및 자동 잠금/해제 |
| **보안성** | 물리적 잠금으로 도난 방지 |
| **효율성** | 실시간 주차 현황 모니터링 |
| **확장성** | BLE Mesh로 대규모 주차장 지원 |

### 4.2 BLE Mesh 프로토콜 구성

| 계층 | 프로토콜 | 기능 |
|------|----------|------|
| **Application** | Generic OnOff Model | 잠금/해제 제어 |
| **Application** | Sensor Model | 자전거 감지 상태 |
| **Application** | Vendor Model | 커스텀 명령 (슬롯 번호 등) |
| **Transport** | Access Layer | 메시지 암호화/인증 |
| **Network** | Network Layer | 메시지 라우팅/릴레이 |
| **Bearer** | ADV Bearer | BLE 광고 패킷 기반 전송 |

## 5. 기술 사양

### 5.1 슬롯 컨트롤러 사양

| 항목 | 사양 |
|------|------|
| **MCU** | Nordic nRF52832 (ARM Cortex-M4F) |
| **무선** | BLE 5.0, Mesh 지원 |
| **전원** | 3.7V 리튬이온 배터리 (3,000mAh) |
| **전원 회로** | 고효율 DC-DC (>90%) |
| **대기 전류** | < 10μA |
| **동작 전류** | < 50mA (통신 시) |
| **배터리 수명** | 2년 이상 (1일 10회 동작 기준) |
| **동작 온도** | -20°C ~ +60°C |
| **방수 등급** | IP65 |
| **크기** | 80 x 80 x 25 mm |

### 5.2 잠금장치 사양

| 항목 | 사양 |
|------|------|
| **타입** | 전자석 솔레노이드 |
| **구동 전압** | 12V DC |
| **구동 시간** | < 100ms |
| **잠금력** | > 500N |
| **내구성** | 100,000회 이상 |
| **동작 온도** | -20°C ~ +60°C |
| **방수 등급** | IP54 |

### 5.3 통신 사양

| 항목 | 사양 |
|------|------|
| **프로토콜** | Bluetooth Mesh (Bluetooth SIG) |
| **주파수** | 2.4GHz ISM Band |
| **통신 거리** | 최대 30m (실내), 50m (실외) |
| **네트워크 규모** | 최대 32,767 노드 |
| **메시지 지연** | < 100ms (3홉 이내) |
| **암호화** | AES-128-CCM |
| **인증** | Device Key, Network Key |
