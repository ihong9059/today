# Military Smart Phone Charging System
## 군용 8채널 스마트폰 충전 보관 시스템

---

## 1. 시스템 개요

### 1.1 제품 소개

군 병영 내 병사들의 스마트폰을 안전하게 보관하고 충전하는 **8채널 스마트 충전 보관함** 시스템입니다.
RTC(실시간 시계)를 통해 휴식 시간을 자동 판별하고, 관리자 RFID 인증으로 보안을 강화하며,
개별 잠금장치로 스마트폰을 안전하게 보관합니다.

### 1.2 적용 분야
- 군 병영 생활관
- 훈련소 휴게실
- 군사 시설 대기실
- 기타 보안이 필요한 스마트폰 보관 장소

### 1.3 핵심 기능

| 기능 | 설명 |
|------|------|
| **8채널 충전** | 8대의 스마트폰 동시 충전 |
| **시간 기반 제어** | RTC로 휴식 시간에만 사용 허용 |
| **연결 감지** | ADC로 스마트폰 연결 상태 확인 |
| **개별 잠금** | 슬롯별 솔레노이드 잠금장치 |
| **RFID 인증** | 관리자 카드로 시스템 운영 |
| **UV 살균** | UV LED로 보관 중 살균 (옵션) |
