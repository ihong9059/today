### 5.2 주요 AT 명령어

#### 기본 제어

```
AT                      # 모듈 응답 확인
AT+CPIN?                # SIM 카드 상태 확인
AT+CSQ                  # 신호 강도 확인
AT+CREG?                # 네트워크 등록 상태
AT+CGDCONT=1,"IP","<APN>"  # APN 설정
```

#### 전원 관리

```
AT+QSCLK=1              # 슬립 모드 활성화
AT+QSCLK=0              # 슬립 모드 비활성화
AT+CFUN=0               # 최소 기능 모드 (RF/SIM 비활성화)
AT+CFUN=1               # 전체 기능 모드
```

#### TCP/IP 통신

```
AT+QIOPEN=<contextID>,<connectID>,"TCP","<IP>",<port>  # TCP 연결
AT+QISEND=<connectID>,<length>                         # 데이터 전송
AT+QIRD=<connectID>,<length>                           # 데이터 수신
AT+QICLOSE=<connectID>                                 # 연결 종료
```

#### MQTT 통신

```
AT+QMTCFG="recv/mode",<client_idx>,<msg_recv_mode>  # MQTT 설정
AT+QMTOPEN=<client_idx>,"<broker>",<port>           # MQTT 브로커 연결
AT+QMTCONN=<client_idx>,"<clientID>"                # MQTT 클라이언트 연결
AT+QMTSUB=<client_idx>,<msgID>,"<topic>",<qos>      # 토픽 구독
AT+QMTPUB=<client_idx>,<msgID>,<qos>,<retain>,"<topic>"  # 메시지 발행
AT+QMTDISC=<client_idx>                              # MQTT 연결 해제
AT+QMTCLOSE=<client_idx>                             # MQTT 세션 종료
```

#### HTTP 통신

```
AT+QHTTPCFG="contextid",<contextID>        # HTTP 컨텍스트 설정
AT+QHTTPCFG="sslctxid",<sslctxID>           # SSL 컨텍스트 설정
AT+QHTTPURL=<url_length>,<timeout>          # URL 설정
AT+QHTTPGET=<timeout>                       # HTTP GET 요청
AT+QHTTPPOST=<data_length>,<timeout>        # HTTP POST 요청
AT+QHTTPREAD=<timeout>                      # 응답 읽기
```

## 14. 결론

QUECTEL EG915 시리즈는 LTE Cat1/Cat1 bis 기반의 M2M 및 IoT 애플리케이션에 최적화된 모듈입니다.

**주요 특징:**
- **컴팩트한 크기**: 23.6 × 19.9 × 2.4 mm
- **글로벌 커버리지**: 지역별 모델로 전 세계 네트워크 지원
- **비용 효율성**: Cat4 대비 저렴한 비용
- **폭넓은 호환성**: EG91/EG95/BG95/BG96과 핀 호환
- **다양한 프로토콜**: MQTT, HTTP, TCP/UDP, SSL 등 지원
- **저전력 설계**: 슬립 모드 및 최소 기능 모드 지원

POS, 스마트 미터링, 자산 추적, 산업 자동화 등 중저속 데이터 전송이 필요한 IoT 애플리케이션에 이상적인 선택입니다.
