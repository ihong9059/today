## 설명 (Description)

```
스마트폰 하나로 집안의 온도를 확인하고, 멀리 있는 기기를 제어한다면 어떨까요?

이제 여러분도 할 수 있습니다!
ESP32 하나면 충분합니다.

이 강좌는 UTTEC Lab에서 제작했습니다.
AI와 함께 배우는 새로운 방식의 IoT 프로그래밍 강좌입니다.

━━━━━━━━━━━━━━━━━━━━━━

📌 이 강좌에서 배우는 내용

🔹 초급 (15일) - 하드웨어 기초
• LED 제어 (켜고 끄기, PWM 밝기 조절)
• 버튼 입력, 부저 멜로디 연주
• I2C 통신 (OLED 디스플레이, 온습도 센서)
• Wi-Fi 연결, 간단한 웹 서버 만들기

🔹 중급 (30일) - 스마트폰 연동
• HTML, CSS, JavaScript 기초
• 스마트폰에서 LED 제어하기
• 센서 데이터 실시간 그래프로 보기
• AJAX, WebSocket 실시간 통신
• OTA 무선 업데이트

🔹 고급 (45일) - 완전한 IoT 시스템
• Node.js PC 서버 구축
• 데이터베이스에 센서 데이터 저장
• 웹 대시보드에서 여러 기기 모니터링
• 사용자 인증, 푸시 알림, 데이터 내보내기
```
