# ESP32 Arduino IoT 강좌 소개 동영상 시나리오

**영상 제목:** ESP32로 시작하는 IoT 프로그래밍 - 스마트폰으로 세상을 제어하세요!
**목표 길이:** 약 5-6분
**대상:** 프로그래밍 입문자, 아두이노에 관심있는 학생/취미가

### 2. ESP32 소개 (0:30 - 1:30)
**[화면]** ESP32 보드 클로즈업, 주요 부품 하이라이트

**[나레이션]**
> "ESP32는 Wi-Fi와 블루투스가 내장된 강력한 마이크로컨트롤러입니다."
>
> "손바닥보다 작은 이 보드 하나에, LED, 부저, 온습도 센서, OLED 디스플레이, 그리고 LoRa 장거리 통신 모듈까지 모두 연결되어 있습니다."
>
> "프로그래밍이 처음이어도 괜찮습니다. 아두이노 IDE를 사용하면 쉽게 코드를 작성하고 업로드할 수 있습니다."

**[화면 전환]** 교육보드 회로도 간략 표시
- LED 3개 (빨강, 노랑, 파랑)
- 부저 2개 (비프, 멜로디)
- 스위치 1개
- OLED 디스플레이
- 온습도 센서 (AHT20)
- LoRa 모듈

---

### 3. 커리큘럼 소개 - 초급 (1:30 - 2:30)
**[화면]** "초급 15일" 타이틀 → 실습 화면 캡처

**[나레이션]**
> "초급 과정은 15일 동안 하드웨어의 기본 동작을 배웁니다."
>
> "LED를 켜고 끄는 것부터 시작해서, PWM으로 밝기를 조절하고, 버튼 입력을 받고, 부저로 멜로디를 연주합니다."
>
> "I2C 통신으로 OLED 디스플레이에 글자를 출력하고, 온습도 센서에서 데이터를 읽어옵니다."
>
> "그리고 Wi-Fi에 연결해서 간단한 웹 서버를 만들어봅니다. 스마트폰 브라우저로 ESP32에 접속해서 센서 데이터를 확인하는 것까지가 초급 과정입니다."

**[화면 요소]**
- Day 1-3: LED 제어
- Day 4-6: 스위치, 부저
- Day 7-9: OLED, 온습도 센서
- Day 10-15: Wi-Fi, 웹 서버

---

### 4. 커리큘럼 소개 - 중급 (2:30 - 3:30)
**[화면]** "중급 30일" 타이틀 → 스마트폰 앱 화면

**[나레이션]**
> "중급 과정에서는 스마트폰과의 연동을 본격적으로 배웁니다."
>
> "HTML, CSS, JavaScript를 배워서 예쁜 웹 앱을 만들고, 스마트폰에서 LED를 제어하고 센서 데이터를 실시간 그래프로 확인합니다."
>
> "AJAX와 WebSocket으로 페이지 새로고침 없이 실시간 통신을 구현하고, 파일 시스템을 활용해서 설정을 저장합니다."
>
> "OTA 업데이트로 USB 연결 없이 무선으로 코드를 업로드하는 것도 배웁니다."

**[화면 요소]**
- 모바일 최적화 UI
- 실시간 그래프
- 터치로 LED 제어
- 알림 시스템

---

### 5. 커리큘럼 소개 - 고급 (3:30 - 4:30)
**[화면]** "고급 45일" 타이틀 → 시스템 구성도

**[나레이션]**
> "고급 과정에서는 PC까지 연동해서 완전한 IoT 시스템을 구축합니다."
>
> "Node.js로 PC 서버를 만들고, 데이터베이스에 센서 데이터를 저장합니다. 웹 대시보드에서 여러 ESP32 기기를 한눈에 모니터링하고 제어할 수 있습니다."
>
> "사용자 인증, 푸시 알림, 데이터 내보내기 같은 실무 기능도 배웁니다."
>
> "그리고 LoRa 모듈을 활용해서 Wi-Fi가 없는 먼 거리까지 데이터를 전송하는 방법도 배웁니다. 여러 센서 노드에서 데이터를 수집해서 게이트웨이를 통해 서버로 전송하는, 진짜 IoT 시스템을 만들어봅니다."

**[화면 요소]**
- 시스템 구성도: 센서노드 → 게이트웨이 → PC서버 → 스마트폰
- PC 대시보드 화면
- LoRa 통신 개념도

## 필요한 영상 소스

### B-roll (보조 영상)
1. ESP32 보드 클로즈업
2. LED 깜빡이는 장면
3. OLED에 텍스트 표시되는 장면
4. 스마트폰으로 웹 페이지 접속 화면
5. PC 대시보드 화면
6. 코드 작성 화면 (Arduino IDE)
7. 브레드보드/배선 없이 깔끔한 교육보드

### 그래픽/애니메이션
1. 커리큘럼 타임라인 (초급→중급→고급)
2. 시스템 구성도 (ESP32-스마트폰-PC 연결)
3. 핀 배치도
4. 텍스트 오버레이 (각 파트 제목)

## 영상 스타일 가이드

- **톤:** 친근하고 쉬운 설명, 기술적 용어는 풀어서 설명
- **페이스:** 빠르지 않게, 초보자가 따라올 수 있도록
- **음악:** 밝고 활기찬 배경음악 (너무 시끄럽지 않게)
- **색상:** 파란색 계열 (기술/신뢰), 주황색 포인트 (활기)
