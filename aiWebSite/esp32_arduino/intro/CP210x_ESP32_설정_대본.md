## 2. Board Manager에서 ESP32 보드 설치

먼저 Arduino IDE에서 ESP32 보드를 사용할 수 있도록 설정하겠습니다.
Board Manager를 통해 보드 패키지를 설치하면 됩니다.

Arduino IDE를 실행합니다.

왼쪽을 보시면 세로로 아이콘들이 있습니다.
그 중에서 칩 모양의 아이콘이 Board Manager입니다.
이 아이콘을 클릭하세요.

Board Manager 패널이 왼쪽에 열립니다.
검색창에 "ESP32"를 입력합니다.

검색 결과에서 "esp32 by Espressif Systems"를 찾습니다.
이것이 공식 ESP32 보드 패키지입니다.

"설치" 버튼을 클릭합니다.

설치에는 보통 2분에서 3분 정도 걸립니다.
인터넷 속도에 따라 다를 수 있으니 잠시 기다려 주세요.

설치가 완료되면 "INSTALLED"라고 표시됩니다.
이제 ESP32 보드를 사용할 준비가 되었습니다!

보드를 선택해 보겠습니다.
상단 메뉴에서 "도구"를 클릭합니다.
"보드"에 마우스를 올리면 "ESP32 Arduino"라는 새로운 항목이 생겼습니다.
여기서 "ESP32 Dev Module"을 선택합니다.

이것은 가장 범용적인 ESP32 보드 설정입니다.
대부분의 ESP32 보드에서 잘 작동합니다.

---

## 3. ESP32 연결 및 포트 확인

이제 실제로 ESP32 보드를 연결해 보겠습니다.

USB 케이블을 준비합니다.
ESP32 보드에 따라 Micro USB 또는 USB-C 타입이 필요합니다.
본인의 보드에 맞는 케이블을 사용하세요.

케이블로 ESP32 보드를 컴퓨터에 연결합니다.

연결이 잘 되었는지 확인하기 위해 장치 관리자를 엽니다.
윈도우 키와 X키를 동시에 누르고, "장치 관리자"를 선택합니다.

장치 관리자에서 "포트 (COM & LPT)" 항목을 찾아 확장합니다.

"Silicon Labs CP210x USB to UART Bridge"라는 항목이 보이면 성공입니다!
이것은 Step 2에서 설치한 드라이버 덕분입니다.

괄호 안에 COM 포트 번호가 표시됩니다.
예를 들어 COM3, COM4 같은 식입니다.
이 번호를 기억해 두세요.

만약 포트 항목이 없거나 CP210x가 안 보인다면,
USB 케이블을 다른 포트에 연결해 보거나,
Step 2의 드라이버를 다시 설치해 보세요.

이제 Arduino IDE로 돌아갑니다.

상단 메뉴에서 "도구"를 클릭합니다.
"포트"에 마우스를 올립니다.
방금 확인한 COM 포트를 선택합니다.

포트 선택이 완료되었습니다!

## 4. 테스트 코드 열기 및 업로드

이제 Step 3에서 미리 다운로드해 둔 테스트 코드를 사용할 차례입니다.

다운로드 폴더로 이동합니다.
ESP32_Serial_Test.zip 파일을 찾아 압축을 해제합니다.

혹시 아직 다운로드하지 않으셨다면,
uttec-edu.duckdns.org에서 C-ESP32 초급 페이지로 이동하시고,
개발환경 설정 가이드의 Step 3에서 다운로드하실 수 있습니다.

압축을 풀면 ESP32_Serial_Test 폴더가 생깁니다.
폴더를 열고 ESP32_Serial_Test.ino 파일을 더블클릭합니다.
Arduino IDE가 자동으로 열리면서 코드가 표시됩니다.

코드를 잠깐 살펴보겠습니다.

Serial.begin(115200)은 시리얼 통신 속도를 115200으로 설정합니다.
Serial.println()은 시리얼 모니터에 메시지를 출력합니다.
loop() 함수에서는 1초마다 카운트가 증가하면서 메시지를 출력합니다.

이 코드가 정상적으로 실행되면, 개발환경 설정이 완료된 것입니다!

업로드하기 전에, 보드와 포트 설정을 한번 더 확인합니다.
도구 메뉴를 클릭해서, 보드가 ESP32 Dev Module로 되어 있는지,
포트가 올바른 COM 포트로 선택되어 있는지 확인하세요.

이제 이 코드를 ESP32에 업로드해 보겠습니다.

왼쪽 위에 있는 업로드 버튼을 클릭합니다.
오른쪽 화살표 모양 버튼입니다.

먼저 컴파일이 진행됩니다.
"컴파일 중"이라는 메시지가 표시됩니다.

컴파일이 완료되면 업로드가 시작됩니다.
하단에 "Connecting"이라는 메시지가 나타납니다.

만약 연결이 안 되면, ESP32 보드의 BOOT 버튼을 누른 상태에서
업로드를 다시 시도해 보세요.
"Connecting" 메시지가 나타나면 BOOT 버튼을 떼면 됩니다.

업로드가 완료되면 "Done uploading"이라는 메시지가 표시됩니다.

이제 시리얼 모니터를 열어서 결과를 확인합니다.
상단 메뉴에서 "도구"를 클릭하고 "시리얼 모니터"를 선택합니다.
단축키는 Ctrl + Shift + M입니다.

시리얼 모니터 창이 열리면, 오른쪽 아래에서 통신 속도를 확인합니다.
115200 baud로 설정되어 있어야 합니다.

화면에 메시지가 보이시나요?

"ESP32 시리얼 통신 테스트!"
"UTTEC Edu - C-ESP32 초급 과정"
"축하합니다! ESP32 개발환경 설정이 완료되었습니다!"

그리고 1초마다 카운트가 증가하면서
"카운트: 1 - Hello from ESP32!"
"카운트: 2 - Hello from ESP32!"
이렇게 메시지가 반복해서 출력됩니다.

이 화면이 보인다면 성공입니다!

축하합니다! ESP32 개발환경 설정이 모두 완료되었습니다!

## 5. 문제 해결 팁

혹시 문제가 생겼다면 몇 가지 확인해 보세요.

첫째, 포트가 안 보인다면,
Step 2의 CP210x 드라이버를 다시 설치하거나, 다른 USB 포트에 연결해 보세요.
USB 허브보다는 컴퓨터에 직접 연결하는 것이 좋습니다.

둘째, 업로드가 실패한다면,
ESP32의 BOOT 버튼을 누른 상태에서 업로드를 시도하세요.
"Connecting" 메시지가 나타나면 BOOT 버튼을 떼면 됩니다.

셋째, 시리얼 출력이 이상하게 보인다면,
시리얼 모니터의 통신 속도가 코드와 일치하는지 확인하세요.
우리는 115200으로 설정했습니다.
또는 ESP32 보드의 EN 버튼, 리셋 버튼을 한번 눌러보세요.

넷째, Board Manager에서 ESP32가 검색이 안 된다면,
보드 매니저 URL이 정확히 입력되었는지 확인하세요.
인터넷 연결 상태도 확인해 보세요.
