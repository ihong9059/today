- **예상 길이**: 8-10분
- **전제조건**:
  - Arduino IDE 설치 완료 (Step 1)
  - CP210x 드라이버 설치 완료 (Step 2)
  - Serial 테스트 코드 다운로드 완료 (Step 3)

## 시나리오 구성

### 1. 오프닝 (30초)
**장면**: 타이틀 화면 + ESP32 보드 실물 이미지

**내용**:
- 동영상 소개
- 사전 준비 확인:
  - Arduino IDE 설치됨 (Step 1 완료)
  - CP210x 드라이버 설치됨 (Step 2 완료)
  - Serial 테스트 코드 다운로드됨 (Step 3 완료)
- 이 영상에서 할 것:
  1. Board Manager에서 ESP32 보드 패키지 설치
  2. ESP32 보드 연결 및 포트 확인
  3. 테스트 코드 업로드 및 시리얼 통신 확인

---

### 2. Board Manager에서 ESP32 보드 설치 (2분 30초)
**장면**: Arduino IDE 화면

**단계**:
1. **Board Manager 아이콘으로 열기**
   - Arduino IDE 실행
   - 왼쪽 사이드바에서 칩 모양 아이콘 (Board Manager) 클릭
   - Board Manager 패널이 왼쪽에 열림

2. **ESP32 보드 패키지 검색 및 설치**
   - 검색창에 "ESP32" 입력
   - "esp32 by Espressif Systems" 찾기
   - 최신 버전 확인 후 "설치" 버튼 클릭
   - 설치 완료까지 대기 (약 2-3분 소요)
   - "INSTALLED" 표시 확인

3. **ESP32 보드 선택**
   - 도구 → 보드 → ESP32 Arduino
   - "ESP32 Dev Module" 선택 (가장 범용적)

---

### 3. ESP32 연결 및 포트 확인 (1분 30초)
**장면**: ESP32 보드 + USB 케이블 연결 장면 → 장치 관리자

**단계**:
1. **ESP32 보드 연결**
   - USB 케이블로 ESP32를 컴퓨터에 연결
   - (Micro USB 또는 USB-C 타입 확인)

2. **포트 인식 확인**
   - 장치 관리자 열기 (Win + X → 장치 관리자)
   - "포트(COM & LPT)" 확장
   - "Silicon Labs CP210x USB to UART Bridge (COMx)" 확인
   - COM 포트 번호 기억 (예: COM3)

3. **Arduino IDE에서 포트 선택**
   - 도구 → 포트 (Tools → Port)
   - 해당 COM 포트 선택

---

### 4. 테스트 코드 열기 및 업로드 (2분 30초)
**장면**: 파일 탐색기 → Arduino IDE 코드 실행 → 시리얼 모니터

**단계**:
1. **미리 다운로드한 테스트 코드 열기**
   - Step 3에서 다운로드한 ESP32_Serial_Test.zip 압축 해제
   - ESP32_Serial_Test 폴더 열기
   - ESP32_Serial_Test.ino 파일 더블클릭 (Arduino IDE로 열림)
   - (아직 다운로드 안 했다면: uttec-edu.duckdns.org → C-ESP32 초급 → Step 3)

2. **코드 확인**
   - 코드 내용 간단히 설명
   - Serial.begin(115200) - 통신 속도 설정
   - Serial.println() - 메시지 출력
   - loop()에서 1초마다 카운트 증가

3. **보드 및 포트 설정 확인**
   - 도구 → 보드 → ESP32 Dev Module 확인
   - 도구 → 포트 → COM 포트 확인

4. **코드 업로드**
   - 업로드 버튼 클릭 (→ 화살표)
   - 컴파일 진행 확인
   - "Connecting..." 메시지 확인
   - 업로드 완료 메시지 확인

5. **시리얼 모니터에서 결과 확인**
   - 도구 → 시리얼 모니터 (또는 Ctrl+Shift+M)
   - 통신 속도 115200 baud 설정
   - 출력 메시지 확인:
     - "ESP32 시리얼 통신 테스트!"
     - "축하합니다! ESP32 개발환경 설정이 완료되었습니다!"
     - "카운트: 1 - Hello from ESP32!" (1초마다 증가)

---

### 5. 문제 해결 팁 (1분)
**장면**: 텍스트 오버레이 + 아이콘

**내용**:
- **포트가 안 보일 때**:
  - CP210x 드라이버 재설치 (Step 2 다시)
  - 다른 USB 포트 시도
  - USB 허브 대신 직접 연결

- **업로드 실패 시**:
  - BOOT 버튼 누른 상태에서 업로드 시도
  - "Connecting..." 표시되면 버튼 떼기

- **시리얼 출력 안 될 때**:
  - 통신 속도(Baud Rate) 115200 확인
  - ESP32 보드의 EN(리셋) 버튼 누르기

- **Board Manager에서 ESP32가 안 보일 때**:
  - 보드 매니저 URL이 정확히 입력되었는지 확인
  - 인터넷 연결 상태 확인
