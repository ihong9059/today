# ESP32 Arduino 초급 강좌 - 진행방법 동영상 시나리오

---

## 영상 개요

- **제목**: ESP32 Arduino 강좌 진행 방법 안내
- **목적**: 학습자가 강좌를 어떻게 진행하는지 이해할 수 있도록 안내
- **예상 길이**: 약 4분 30초
- **대상**: ESP32 Arduino 초급 수강자

---

## 화면 구성

### Scene 1: 인트로 (0:00 ~ 0:20)

**화면**: UTTEC Edu 로고 + "ESP32 Arduino 초급 - 진행방법 안내" 타이틀

**대본**:
```
안녕하세요, UTTEC Edu ESP32 Arduino 강좌에 오신 것을 환영합니다.
이 영상에서는 강좌를 어떻게 진행하는지 안내해 드리겠습니다.

이 강좌는 AI와 함께 배우는 새로운 방식으로 설계되었습니다.
```

---

### Scene 2: 학습 목표 소개 (0:20 ~ 0:45)

**화면**: Day 1 페이지의 "오늘의 학습 목표" 섹션 화면

**대본**:
```
각 Day마다 학습 목표가 주어집니다.

Day 1을 예로 들면,
첫 번째 목표는 "ESP32와 Arduino 이해하기"
두 번째 목표는 "시리얼 모니터로 Hello ESP32 출력하기"
세 번째 목표는 "변수 사용하기" 입니다.

각 목표를 순서대로 완료하면 다음 목표로 진행됩니다.
```

---

### Scene 3: Step 1 - AI에게 질문하기 (0:45 ~ 1:40)

**화면**: Step 1 "AI에게 질문하기" 섹션 확대

**대본**:
```
자, 이제 첫 번째 목표를 진행해 보겠습니다.

목표 아래에 있는 Step 1 "AI에게 질문하기" 섹션을 보시면,
실습 내용이 텍스트 박스에 미리 작성되어 있습니다.

이 내용을 AI에게 질문할 겁니다.

먼저 "복사" 버튼을 클릭하면 내용이 복사됩니다.
```

**화면**: 복사 버튼 클릭 → "복사됨!" 표시

**대본**:
```
또는 아래에 있는 AI 서비스 버튼을 직접 클릭하면,
자동으로 내용이 복사되고 해당 AI 사이트가 새 창으로 열립니다.

ChatGPT, Claude, Gemini, Copilot 중 하나를 선택해서 클릭하세요.
```

**화면**: ChatGPT 버튼 클릭 → ChatGPT 사이트 열림

**대본**:
```
AI 사이트가 열리면, 입력창에 Ctrl+V를 눌러 붙여넣기 하세요.
그리고 전송 버튼을 클릭하면 AI가 답변을 생성합니다.
```

---

### Scene 4: Step 2 - AI 결과 붙여넣기 (1:40 ~ 2:40)

**화면**: AI 답변 화면 → Step 2 "AI 결과 붙여넣기" 섹션

**대본**:
```
AI가 답변을 완료하면, 답변 내용을 복사합니다.
AI 화면에서 답변 내용을 드래그하고 Ctrl+C를 눌러 복사하세요.

다시 강좌 페이지로 돌아와서,
Step 2 "AI 결과 붙여넣기" 텍스트 박스에 Ctrl+V로 붙여넣습니다.
```

**화면**: 텍스트 박스에 AI 결과 붙여넣는 모습

**대본**:
```
붙여넣기가 완료되면, "다른 이름으로 저장" 버튼을 클릭하여
Arduino 프로젝트 폴더에 .ino 파일로 저장합니다.
```

**화면**: 파일 저장 다이얼로그 → Arduino IDE에서 파일 열기

**대본**:
```
Arduino IDE에서 저장된 파일을 열어 내용을 확인합니다.
그리고 ESP32 보드에 업로드 버튼을 눌러 코드를 업로드하세요.
```

**화면**: Arduino IDE 업로드 버튼 클릭 → 업로드 완료

**대본**:
```
업로드가 완료되면 시리얼 모니터를 열어서
결과를 직접 눈으로 확인해 보세요.
```

---

### Scene 5: Step 3 - 학습 확인 퀴즈 (2:40 ~ 3:15)

**화면**: Step 3 "학습 확인 퀴즈" 섹션

**대본**:
```
AI의 답변을 확인하고 코드도 실행해 봤다면,
Step 3 학습 확인 퀴즈를 풀어봅니다.

문제를 읽고 정답이라고 생각하는 보기를 클릭한 후,
"정답 확인" 버튼을 클릭하세요.
```

**화면**: 퀴즈 정답 선택 → 정답 확인 클릭 → "정답입니다!" 표시

**대본**:
```
정답을 맞추면 자동으로 다음 목표로 이동합니다.

만약 틀렸다면, "다시 시도" 버튼을 클릭해서 다시 풀 수 있습니다.
AI의 답변을 다시 읽어보고 정답을 찾아보세요.
```

---

### Scene 6: 다음 목표로 이동 (3:15 ~ 3:40)

**화면**: 다음 목표 섹션이 활성화되는 모습

**대본**:
```
첫 번째 목표의 퀴즈를 맞추면,
첫 번째 목표는 "완료" 상태로 바뀌고,
두 번째 목표가 활성화됩니다.

같은 방식으로 Step 1, 2, 3을 반복하며
나머지 목표들도 완료해 주세요.
```

---

### Scene 7: Day 완료 (3:40 ~ 4:05)

**화면**: 모든 목표 완료 → "Day 1 학습 완료!" 축하 메시지

**대본**:
```
모든 목표를 완료하면,
"Day 1 학습 완료!" 축하 메시지가 나타납니다.

"다음 강의" 버튼을 클릭하면 Day 2로 이동할 수 있습니다.

매일 하나의 Day를 완료하면,
15일 만에 ESP32 Arduino 초급 과정을 마스터할 수 있습니다!
```

## Python 강좌와의 차이점

| 항목 | Python 강좌 | ESP32 Arduino 강좌 |
|------|-------------|-------------------|
| 파일 확장자 | .py, .txt | .ino |
| 실행 환경 | VS Code | Arduino IDE |
| 결과 확인 | 터미널 실행 | 시리얼 모니터 |
| 업로드 과정 | 없음 | ESP32에 업로드 필요 |

| 목표 개수 (Day 1) | 2개 | 3개 |
