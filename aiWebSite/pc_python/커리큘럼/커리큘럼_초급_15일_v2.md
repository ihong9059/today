## Day 5: 반복하기 (for문)

### 목표 1: 정해진 횟수만큼 반복하기

#### 오늘 배울 것
같은 작업을 여러 번 하려면 **for문**과 **range()**를 사용합니다.

#### AI에게 요청하기
```
"화이팅!"을 5번 출력하는 Python 코드 만들어줘
```

#### 코드 이해하기
- `for i in range(5):`는 5번 반복하는구나
- range(5)는 0, 1, 2, 3, 4를 차례로 주는구나
- i에는 현재 몇 번째인지 숫자가 들어가는구나
- 들여쓰기한 코드가 반복되는구나

#### 수정해보기
1. 10번 반복하도록 바꿔보세요
2. "1번째 화이팅!", "2번째 화이팅!" 처럼 번호도 함께 출력해보세요

#### 정리
- 반복: `for 변수 in range(횟수):`
- range(n)은 0부터 n-1까지

---

### 목표 2: 1부터 N까지 합 구하기

#### 오늘 배울 것
반복문으로 **누적 계산**을 할 수 있습니다.

#### AI에게 요청하기
```

```

#### 코드 이해하기

- `range(1, 101)`은 1부터 100까지구나 (끝 숫자는 포함 안 됨!)
- 반복하면서 변수에 계속 더하는구나

#### 수정해보기
1. 1부터 10까지의 합을 구해보세요
2. 사용자가 숫자를 입력하면 1부터 그 숫자까지의 합을 구하도록 바꿔보세요

#### 정리
- range(시작, 끝): 끝 숫자는 포함 안 됨
- 누적: 변수를 0으로 초기화 후 반복문에서 더하기
