## Day 1: 2차원 리스트와 정렬

#### AI에게 요청하기

## Day 2: 세트 (집합)

### 목표 1: 중복 제거하기

#### 오늘 배울 것
**세트(Set)**는 중복을 허용하지 않는 자료형입니다. 집합 연산도 가능합니다.

#### AI에게 요청하기
```
리스트에서 중복된 값을 제거하고 유일한 값만 남기는 Python 코드 만들어줘.
예시: [1, 2, 2, 3, 3, 3, 4] → [1, 2, 3, 4]
```

#### 코드 이해하기
- `set(리스트)`로 중복이 제거되는구나
- 세트는 순서가 없구나
- 다시 리스트로 바꾸려면 `list()`를 쓰는구나
- `{값1, 값2}`로 세트를 직접 만들 수도 있구나

#### 수정해보기
1. 문자열 리스트에서 중복을 제거해보세요
2. 결과를 정렬된 상태로 출력해보세요

#### 정리
- 세트: 중복 없는 집합
- 중복 제거: `list(set(리스트))`

---

### 목표 2: 집합 연산하기

#### 오늘 배울 것
세트로 **합집합, 교집합, 차집합**을 구할 수 있습니다.

#### AI에게 요청하기
```
두 반의 학생 명단이 있을 때,
1. 두 반 모두에 있는 학생 (교집합)
2. 두 반 중 한 곳에라도 있는 학생 (합집합)
3. A반에만 있는 학생 (차집합)
을 구하는 Python 코드 만들어줘
```

#### 코드 이해하기
- `&` 또는 `.intersection()`은 교집합이구나
- `|` 또는 `.union()`은 합집합이구나
- `-` 또는 `.difference()`는 차집합이구나
- 집합 연산으로 데이터 비교가 쉬워지는구나

#### 수정해보기
1. B반에만 있는 학생도 구해보세요
2. 한쪽에만 있는 학생(대칭차집합)을 구해보세요

#### 정리
- 교집합: `A & B`, 합집합: `A | B`, 차집합: `A - B`
- 대칭차집합: `A ^ B`

- 안전한 접근: `.get("키", 기본값)`

## Day 4: 문자열 포맷팅

### 목표 1: f-string 활용하기

#### 오늘 배울 것
**f-string**은 가장 간편하고 현대적인 문자열 포맷팅 방법입니다.

#### AI에게 요청하기
```
이름, 나이, 점수 변수가 있을 때 f-string을 사용해서
"홍길동님(25세)의 점수는 85.5점입니다." 형태로 출력하는 코드 만들어줘
```

#### 코드 이해하기
- `f"문자열 {변수}"` 형태로 변수를 삽입하는구나
- 중괄호 안에 변수 이름을 직접 쓰는구나

#### 정리
- f-string: `f"텍스트 {변수}"`
- 표현식 가능: `f"{점수 * 2}"`

### 목표 2: 숫자 포맷팅

#### 오늘 배울 것
숫자를 **소수점, 천단위 콤마** 등 원하는 형식으로 출력할 수 있습니다.

#### AI에게 요청하기

- 천단위 콤마: `{숫자:,}`
- 소수점 고정: `{숫자:.2f}`

#### 오늘 배울 것

- 정규표현식: `import re`
- 패턴 찾기: `re.findall(패턴, 문자열)`

### 목표 2: 패턴으로 검증하기

#### 오늘 배울 것
입력값이 특정 형식에 맞는지 **검증**할 수 있습니다.

#### AI에게 요청하기
```
사용자가 입력한 비밀번호가 다음 조건을 만족하는지 검사하는 코드 만들어줘:
- 8자 이상
- 숫자 포함
- 영문자 포함
```

#### 코드 이해하기

- `\d`는 숫자, `[a-zA-Z]`는 영문자구나
- 여러 조건을 각각 검사해서 조합하는구나
- `{8,}`는 8개 이상이라는 의미구나

#### 수정해보기

2. 어떤 조건이 부족한지 알려주는 메시지를 출력해보세요

#### 정리
- 검색: `re.search(패턴, 문자열)`
- 숫자: `\d`, 영문자: `[a-zA-Z]`
- 개수: `{최소,최대}`

#### 코드 이해하기

#### 정리

### 목표 2: 정렬 알고리즘 이해하기

#### 오늘 배울 것
데이터를 순서대로 배열하는 **정렬 알고리즘**을 이해합니다.

#### AI에게 요청하기

- Python 내장 정렬은 훨씬 빠른 알고리즘(Timsort)을 쓰는구나
- `time` 모듈로 시간을 측정할 수 있구나

#### 정리
- 버블/선택/삽입 정렬: O(n²) - 느림
- Python 내장 정렬: O(n log n) - 빠름
- 실무에서는 내장 정렬 사용!
