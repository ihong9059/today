## 학습 권장 사항

### 매일 학습 시간
- **권장:** 30분 ~ 1시간
- **최소:** 20분

### 효과적인 학습 방법
1. **AI 질문:** Step 1에서 개념 이해
2. **파일 저장:** 설명은 .md, 코드는 .py로 저장
3. **직접 실행:** VS Code에서 코드 실행
4. **퀴즈:** Step 3로 이해도 확인
5. **복습:** 어려운 부분은 다시 AI에게 질문

### 어려울 때
- AI에게 더 쉽게 설명해달라고 요청
- 예제 코드를 직접 수정해보기
- 이전 Day 복습하기
