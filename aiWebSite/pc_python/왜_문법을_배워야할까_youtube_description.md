## 설명 (Description)

```
"AI가 코드 다 짜주는데, 내가 왜 배워야 해?"
이 질문에 대한 명쾌한 답을 알려드립니다!

ChatGPT, Claude 같은 AI가 코드를 만들어주는 시대,
그래도 프로그래밍 문법을 배워야 하는 이유가 있습니다.

━━━━━━━━━━━━━━━━━━━━━━

📌 이 영상에서 배우는 내용

1️⃣ AI는 틀릴 수 있다
   - AI가 만든 코드에 에러가 있을 때, 직접 고칠 수 있어요

2️⃣ 내가 원하는 대로 수정 가능
   - 반복 횟수 3→5로 바꾸기? 문법 알면 1초!

3️⃣ AI와 더 잘 대화할 수 있다
   - 프로그래밍 용어를 알면 정확한 질문 → 정확한 답변

4️⃣ 코드를 레고처럼 조립할 수 있다
   - if, for, print = 블록! 조합해서 나만의 프로그램 만들기

5️⃣ 미래를 위한 준비
   - AI를 도구로 활용하는 능력 + 논리적 사고력

━━━━━━━━━━━━━━━━━━━━━━
```
