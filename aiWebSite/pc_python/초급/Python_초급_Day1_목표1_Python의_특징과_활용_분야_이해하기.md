## 1) Python은 언제, 누가 만들었나요?

* **만든 사람:** 귀도 반 로섬(Guido van Rossum)
* **시작:** 1989년쯤 개발을 시작했고,
* **처음 공개:** 1991년에 처음 버전을 공개했어요.

파이썬은 “사람이 읽기 쉬운 코드”를 목표로 만든 언어입니다.

---

## 2) Python의 가장 큰 특징 3가지

### (1) 문법이 쉽고, 읽기 편해요

코드가 “영어 문장처럼” 보이도록 설계되어 있어요.

```python
print("Hello, Python!")
```

### (2) 라이브러리(도구상자)가 엄청 많아요

이미 만들어진 기능들이 매우 많아서, 복잡한 것도 빨리 만들 수 있어요.
예: 웹 만들기(Flask/Django), 데이터 분석(pandas), AI(PyTorch/TensorFlow) 등

### (3) 다양한 환경에서 바로 실행돼요(범용성)

Windows / macOS / Linux 등 어디서든 비슷하게 동작하고,
서버·PC·라즈베리파이 등에서도 많이 씁니다.

---

## 3) Python이 사용되는 분야는 어디인가요?

파이썬은 “범용 언어”라서 거의 다 합니다. 대표적으로:

### ✅ AI / 머신러닝

* 이미지 인식, 챗봇, 추천 시스템 등

### ✅ 웹 개발

* 웹사이트, API 서버 (Django, Flask, FastAPI)

### ✅ 데이터 분석 / 시각화

* 엑셀 같은 표 데이터 처리, 그래프 그리기 (pandas, matplotlib)

### ✅ 자동화(업무 효율)

* 파일 정리, 메일/문서 처리, 반복 작업 자동화

### ✅ IoT / 임베디드

* 라즈베리파이 센서 제어, 간단 서버 구축 등

간단 예시(반복 작업 자동화 느낌):

```python
for i in range(5):
    print(i, "번째 작업 실행")
```

## 4) Python이 C, Java보다 배우기 쉬운 이유는?

### (1) 코드가 짧고 직관적이에요

같은 일을 더 적은 코드로 할 수 있어요.

```python
a = 10
b = 20
print(a + b)
```

### (2) “타입 선언”을 처음부터 강요하지 않아요

C/Java는 `int`, `String` 같은 타입을 자주 적어야 하는데,
파이썬은 초보자가 일단 시작하기 쉬워요.

```python
name = "홍길동"
age = 10
print(name, age)
```

### (3) 바로 실행하면서 배울 수 있어요

컴파일(미리 빌드) 과정이 복잡하지 않고, 한 줄씩 실행하며 결과를 보면서 공부하기 좋아요.

## 아주 간단한 종합 예시(입력 → 처리 → 출력)

```python
name = input("이름을 입력하세요: ")
print("안녕하세요,", name, "님!")
```

* `input()` : 사용자 입력 받기
* `print()` : 화면에 출력하기
