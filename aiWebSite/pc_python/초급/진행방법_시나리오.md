# Python(PC) 초급 강좌 - 진행방법 동영상 시나리오

---

## 영상 개요

- **제목**: Python 강좌 진행 방법 안내
- **목적**: 학습자가 강좌를 어떻게 진행하는지 이해할 수 있도록 안내
- **예상 길이**: 약 3분 50초
- **대상**: Python 초급 수강자

---

## 화면 구성

### Scene 1: 인트로 (0:00 ~ 0:15)

**화면**: UTTEC Edu 로고 + "Python(PC) 초급 - 진행방법 안내" 타이틀

**대본**:
```
안녕하세요, UTTEC Edu Python 강좌에 오신 것을 환영합니다.
이 영상에서는 강좌를 어떻게 진행하는지 안내해 드리겠습니다.
```

---

### Scene 2: 학습 목표 소개 (0:15 ~ 0:35)

**화면**: Day 1 페이지의 "오늘의 학습 목표" 섹션 화면

**대본**:
```
각 Day마다 학습 목표가 주어집니다.

Day 1을 예로 들면,
첫 번째 목표는 "Python의 특징과 활용 분야 이해하기"
두 번째 목표는 "첫 번째 Hello World 프로그램 실행하기" 입니다.

각 목표를 순서대로 완료하면 다음 목표로 진행됩니다.
```

---

### Scene 3: Step 1 - AI에게 질문하기 (0:35 ~ 1:25)

**화면**: Step 1 "AI에게 질문하기" 섹션 확대

**대본**:
```
자, 이제 첫 번째 목표를 진행해 보겠습니다.

목표 아래에 있는 Step 1 "AI에게 질문하기" 섹션을 보시면,
실습 내용이 텍스트 박스에 미리 작성되어 있습니다.

이 내용을 AI에게 질문할 겁니다.

먼저 "복사" 버튼을 클릭하면 내용이 복사됩니다.
```

**화면**: 복사 버튼 클릭 → "복사됨!" 표시

**대본**:
```
또는 아래에 있는 AI 서비스 버튼을 직접 클릭하면,
자동으로 내용이 복사되고 해당 AI 사이트가 새 창으로 열립니다.

ChatGPT, Claude, Gemini, Copilot 중 하나를 선택해서 클릭하세요.
```

**화면**: ChatGPT 버튼 클릭 → ChatGPT 사이트 열림

**대본**:
```
AI 사이트가 열리면, 입력창에 Ctrl+V를 눌러 붙여넣기 하세요.
그리고 전송 버튼을 클릭하면 AI가 답변을 생성합니다.
```

---

### Scene 4: Step 2 - AI 결과 붙여넣기 (1:25 ~ 2:15)

**화면**: AI 답변 화면 → Step 2 "AI 결과 붙여넣기" 섹션

**대본**:
```
AI가 답변을 완료하면, 답변 내용을 복사합니다.
AI 화면에서 답변 내용을 드래그하고 Ctrl+C를 눌러 복사하세요.

다시 강좌 페이지로 돌아와서,
Step 2 "AI 결과 붙여넣기" 텍스트 박스에 Ctrl+V로 붙여넣습니다.
```

**화면**: 텍스트 박스에 AI 결과 붙여넣는 모습

**대본**:
```
붙여넣기가 완료되면, "다른 이름으로 저장" 버튼을 클릭하여
VS Code로 연결된 폴더에 파일을 저장합니다.

VS Code에서 저장된 파일을 열어 내용을 확인하고,
코드가 있다면 직접 실행해서 결과를 확인할 수 있습니다.
```

**화면**: VS Code에서 파일 열기 → 코드 실행 화면

**대본**:
```
VS Code는 이미 Python을 실행할 수 있도록 설정되어 있으니,
코드를 실행하여 결과를 직접 눈으로 확인해 보세요.
```

---

### Scene 5: Step 3 - 학습 확인 퀴즈 (2:15 ~ 2:45)

**화면**: Step 3 "학습 확인 퀴즈" 섹션

**대본**:
```
AI의 답변을 확인했다면, Step 3 학습 확인 퀴즈를 풀어봅니다.

문제를 읽고 정답이라고 생각하는 보기를 클릭한 후,
"정답 확인" 버튼을 클릭하세요.
```

**화면**: 퀴즈 정답 선택 → 정답 확인 클릭 → "정답입니다!" 표시

**대본**:
```
정답을 맞추면 자동으로 다음 목표로 이동합니다.

만약 틀렸다면, "다시 시도" 버튼을 클릭해서 다시 풀 수 있습니다.
AI의 답변을 다시 읽어보고 정답을 찾아보세요.
```

---

### Scene 6: 다음 목표로 이동 (2:45 ~ 3:05)

**화면**: 다음 목표 섹션이 활성화되는 모습

**대본**:
```
첫 번째 목표의 퀴즈를 맞추면,
첫 번째 목표는 "완료" 상태로 바뀌고,
두 번째 목표가 활성화됩니다.

같은 방식으로 Step 1, 2, 3을 반복하며
두 번째 목표도 완료해 주세요.
```

---

### Scene 7: Day 완료 (3:05 ~ 3:30)

**화면**: 모든 목표 완료 → "Day 1 학습 완료!" 축하 메시지

**대본**:
```
모든 목표를 완료하면,
"Day 1 학습 완료!" 축하 메시지가 나타납니다.

"다음 강의" 버튼을 클릭하면 Day 2로 이동할 수 있습니다.

매일 하나의 Day를 완료하면,
15일 만에 Python 초급 과정을 마스터할 수 있습니다!
```

---

### Scene 8: 아웃트로 (3:30 ~ 3:50)

**화면**: 요약 정리 화면

**대본**:
```
정리하겠습니다.

1. 학습 목표 확인
2. Step 1: 실습 내용을 복사하여 AI에게 질문
3. Step 2: AI 답변을 붙여넣고 VS Code에 저장하여 확인
4. Step 3: 퀴즈를 풀어 다음 목표로 이동

이 순서대로 진행하시면 됩니다.

그럼 지금 바로 첫 번째 목표부터 시작해 보세요!
UTTEC Edu와 함께하는 Python 학습, 화이팅입니다!
```
