Python 학습을 시작하신 것을 축하드립니다! Python은 문법이 간결하여 프로그래밍 입문자에게 가장 추천되는 언어입니다. 요청하신 내용을 단계별로 정리해 드릴게요.

---

### 1. VS Code에서 .py 파일 만들기

1. **VS Code 실행**: 프로그램을 엽니다.
2. **폴더 열기**: `File` > `Open Folder`를 클릭해 코드를 저장할 폴더를 선택합니다.
3. **파일 생성**: 왼쪽 탐색기(Explorer) 창에서 폴더 이름 옆의 **새 파일 아이콘**을 클릭하거나, `Ctrl + N`을 누릅니다.
4. **저장**: 파일 이름을 `hello.py`와 같이 입력합니다. 확장자가 반드시 **.py**여야 Python 파일로 인식됩니다.

---

### 2. "Hello, World!" 출력 코드

Python에서는 단 한 줄이면 충분합니다.

```python
print("Hello, World!")

```

---

### 3. print() 함수 사용법

`print()` 함수는 화면(터미널)에 데이터를 출력하는 역할을 합니다.

* **문자열 출력**: 큰따옴표(`" "`)나 작은따옴표(`' '`)로 감쌉니다.
* **숫자 출력**: 따옴표 없이 숫자만 입력합니다.
* **여러 값 출력**: 콤마(`,`)를 사용해 여러 내용을 한 번에 출력할 수 있습니다. (기본적으로 띄어쓰기가 포함됩니다.)

```python
print("나이:", 25)  # 출력 결과: 나이: 25

```

---

### 4. VS Code에서 실행하는 방법

두 가지 주요 방법이 있습니다.

* **실행 버튼 (추천)**: 편집기 오른쪽 상단에 있는 **화살표 모양의 'Run Python File' 아이콘**을 클릭합니다.
* **터미널 직접 입력**: 하단 터미널창에 다음과 같이 입력하고 엔터를 누릅니다.
> `python hello.py` (Windows) 또는 `python3 hello.py` (Mac/Linux)



---

### 5. C언어와 Python의 비교

C언어로 같은 프로그램을 만들 때와 비교해 보면 Python이 얼마나 직관적인지 알 수 있습니다.

| 특징 | C언어 | Python |
| --- | --- | --- |
| **코드 길이** | 약 5~6줄 | **단 1줄** |
| **복잡도** | 헤더 선언, 메인 함수, 세미콜론(;) 필요 | **핵심 로직만 작성** |
| **가독성** | 기계적인 구조가 강함 | **영어 문장처럼 읽힘** |

**[C언어 예시]**

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

```

**[Python 예시]**

```python
print("Hello, World!")

```

C언어는 메모리 관리나 데이터 타입을 직접 신경 써야 하지만, Python은 이러한 복잡한 과정을 내부적으로 처리해주어 **개발자가 로직에만 집중**할 수 있게 해줍니다.
