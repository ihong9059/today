| 항목 | 내용 |

| **대상** | 코딩 초보자, 초등학생~중학생, 학부모 |
| **길이** | 약 6분 |
| **목적** | AI 시대에도 프로그래밍 문법 학습이 필요한 이유 설명 |

## 시나리오

### [인트로] (0:00 ~ 0:30)

**화면:**
- ChatGPT, Claude 등 AI 화면
- "Python으로 계산기 만들어줘" 입력 → 코드 생성되는 모습

**나레이션:**
```
요즘 AI한테 물어보면 코드를 뚝딱 만들어줍니다.
"Python으로 계산기 만들어줘" 하면 바로 코드가 나오죠.

그런데 이런 생각이 들 수 있어요.
"AI가 다 해주는데, 내가 왜 문법을 배워야 하지?"

오늘은 이 질문에 대한 답을 찾아보겠습니다.
```

---

### [섹션 1: AI의 한계 - 틀릴 수 있다] (0:30 ~ 1:30)

**화면:**
- AI가 생성한 코드에 빨간 에러 표시
- 실행했는데 오류 나는 화면
- 당황한 표정의 캐릭터

**나레이션:**
```
첫 번째 이유, AI는 틀릴 수 있습니다.

AI가 만들어준 코드를 실행했는데... 에러!
"왜 안 되지?" 하고 당황하게 됩니다.

문법을 모르면 어디가 틀렸는지 알 수 없어요.
AI한테 다시 물어봐도, 또 이상한 답이 올 수 있죠.

예를 들어볼게요.
AI가 이런 코드를 줬다고 해봅시다.

print("안녕하세요)

실행하면 에러가 납니다.
문법을 아는 사람은 바로 알 수 있어요.
"아, 따옴표가 하나 빠졌구나!"

print("안녕하세요")

이렇게 고치면 되는 거죠.

문법을 모르면 이 작은 실수도 찾을 수 없습니다.
```

---

### [섹션 2: 내가 원하는 대로 수정하기] (1:30 ~ 2:30)

**화면:**
- AI가 만든 코드
- "색깔을 바꾸고 싶은데..." 고민하는 모습
- 코드 일부를 수정하는 모습

**나레이션:**
```
두 번째 이유, 내가 원하는 대로 수정할 수 있습니다.

AI가 만들어준 코드가 마음에 안 들 때가 있어요.
"숫자를 3번 반복하는데, 5번으로 바꾸고 싶어"
"글자 색깔을 빨간색으로 바꾸고 싶어"

문법을 알면, 직접 코드를 수정할 수 있습니다.

for i in range(3):
    print("안녕")

여기서 3을 5로 바꾸면?

for i in range(5):
    print("안녕")

5번 반복하게 되죠!

문법을 모르면 매번 AI한테 다시 물어봐야 해요.
"3을 5로 바꿔줘"
"아니, 그게 아니라..."
"다시 해줘..."

시간도 오래 걸리고, 답답하겠죠?
```

---

### [섹션 3: AI와 대화하기] (2:30 ~ 3:30)

**화면:**
- AI에게 질문하는 화면
- 구체적인 질문 vs 막연한 질문 비교
- AI가 더 좋은 답변을 주는 모습

**나레이션:**
```
세 번째 이유, AI와 더 잘 대화할 수 있습니다.

문법을 알면, AI에게 더 정확하게 질문할 수 있어요.

문법을 모르는 사람:
"뭔가 이상해요. 고쳐주세요."

문법을 아는 사람:
"for문에서 range 값을 10으로 바꾸고,
print 안에 변수 i를 넣어주세요."

어떤 질문이 더 정확한 답을 받을까요?

당연히 두 번째죠!

AI도 사람처럼, 질문이 구체적이면 더 좋은 답을 줍니다.
프로그래밍 용어를 알면, AI와 대화가 훨씬 쉬워져요.
```

---

### [섹션 4: 레고 블록처럼 이해하기] (3:30 ~ 4:30)

**화면:**
- 레고 블록 조립하는 모습
- 코드를 블록처럼 분리해서 보여주는 애니메이션
- 블록을 조합해서 프로그램 만드는 모습

**나레이션:**
```
네 번째 이유, 코드를 이해하면 레고처럼 조립할 수 있어요.

레고를 생각해보세요.
처음에는 설명서 보고 그대로 따라 만들죠.
하지만 블록이 어떻게 끼워지는지 알면?
나만의 새로운 작품을 만들 수 있어요!

프로그래밍도 똑같습니다.

if는 "만약 ~라면"
for는 "여러 번 반복해"
print는 "화면에 보여줘"

이런 블록들을 알면,
AI가 만든 코드를 이해할 수 있고,
블록을 조합해서 나만의 프로그램을 만들 수 있어요.

게임 캐릭터가 점프하게 하고 싶다면?
if와 for를 조합하면 되겠구나!

이렇게 생각할 수 있게 됩니다.
```

### [섹션 5: 미래를 위한 준비] (4:30 ~ 5:30)

**화면:**
- 다양한 직업 (개발자, 과학자, 디자이너, 의사 등)
- AI와 함께 일하는 모습
- 코딩으로 문제를 해결하는 모습

**나레이션:**
```
다섯 번째 이유, 미래를 위한 준비입니다.

AI는 점점 더 똑똑해지고 있어요.
그런데 AI를 잘 사용하는 사람과 못 사용하는 사람,
누가 더 잘 될까요?

프로그래밍을 이해하면,
AI가 뭘 잘하고, 뭘 못하는지 알 수 있어요.
AI를 도구로 잘 활용할 수 있게 되죠.

요리사가 칼을 잘 다루듯이,
미래에는 AI를 잘 다루는 능력이 중요해집니다.

그리고 프로그래밍 문법은
"논리적으로 생각하는 방법"을 알려줘요.

1. 문제를 작은 단계로 나누고
2. 순서대로 해결하고
3. 실수를 찾아서 고치고

이런 능력은 어떤 직업을 갖든 도움이 됩니다.
프로그래머가 아니어도요!
```
