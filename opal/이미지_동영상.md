# prompt
## 주재 입력: ai의 필요성
만들고 샆은 주제를 입력해 주세요.

## 주제 탐색
[주제입력]을 조사해서 요약해줘

요약한 내용을 4문단으로 나눠줘.
그리고 각 문단별로 나레이션을 만들어줘

만든 내용은

요약내용1:
나레이션1:

의 형태로 만들어줘.

## 인물태그확장
 {{"type": "in", "path": "a509f56a-9c70-444b-a503-8c0fe0b8b916", "title": "인물생성"}}에 대해 인물을 생성해줘.
가능하면 배경은 없이 인물만 생성해줘.

## 요약1꺼내기
 {{"type": "in", "path": "78912369-9cbb-47e6-bb20-294b702d7e1f", "title": "주제 탐색"}}에서 요약내용1의 텍스트만 가져와

## 이미지 생성1

 {{"type": "in", "path": "be33bc82-fcb4-46a9-b66c-651d005116cd", "title": "인물태그확장"}}의 인물이 자연스럽게 화면에 들어오게 해줘. 의상과 포즈이런것은 부드럽게 수정해줘.
