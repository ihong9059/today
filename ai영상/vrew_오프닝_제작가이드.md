## 2. Vrew 프로젝트 생성

### Step 1: 새 프로젝트 시작
1. Vrew 실행
2. `새로 만들기` 클릭
3. `텍스트로 비디오 만들기` 선택
4. 화면 비율: **16:9 (가로형)** 선택

### Step 2: 프로젝트 설정
- **해상도:** 1920x1080 (Full HD)
- **프레임 레이트:** 30fps

## 3. 대사 입력 (섹션별)

### 섹션 1: 도입 (0:00~0:30)

```
[자막만 표시 - 나레이션 없음]
세상이 빠르게 변하고 있습니다
```

**Vrew 설정:**
- 클립 유형: 자막만 (AI 음성 OFF)
- 배경: AI/기술 관련 스톡 이미지 또는 동영상
- 자막 스타일: 크고 굵은 폰트, 중앙 배치
- 효과: 페이드 인

---

### 섹션 2: 통계 및 AI 시대 (0:30~1:30)

```
ChatGPT가 등장한 지 불과 2년.
이미 우리 삶 곳곳에 AI가 자리 잡았습니다.

전문가들은 말합니다.
현재 직업의 70%가 앞으로 10년 안에 변화할 것이라고.
어떤 직업은 사라지고, 어떤 직업은 새로 생겨날 것입니다.

이 변화는 멈추지 않습니다.
오히려 더 빨라지고 있죠.
```

**Vrew 설정:**
- AI 음성: 남성 차분한 목소리 권장
- 속도: 0.9x ~ 1.0x (여유있게)
- 배경: 통계 그래프, 직업 변화 인포그래픽
- 자막: 하단 중앙, 가독성 좋은 폰트

---

### 섹션 3: 인터뷰 형식 (1:30~2:30)

#### 학부형 인터뷰
```
아이가 중학생인데요, 어떤 진로를 선택해야 할지 정말 막막해요.
제가 알던 직업들이 10년 후에도 있을지 모르겠거든요.
```

**Vrew 설정:**
- AI 음성: 여성 목소리 (40대 느낌)
- 자막 상단: [학부형 인터뷰]
- 배경: 학부형 실루엣 또는 인터뷰 배경

#### 취준생 인터뷰
```
전공을 살려서 취업하려고 하는데요,
AI 때문에 이 분야도 미래가 불안해요.
그렇다고 지금 다른 걸 시작하기엔 늦은 것 같고...
```

**Vrew 설정:**
- AI 음성: 젊은 남성/여성 목소리
- 자막 상단: [취준생 인터뷰]
- 배경: 취업준비생 이미지

#### 내레이션 연결
```
변화의 시대.
우리 아이들은 어떻게 준비해야 할까요?
취업을 준비하는 청년들은 어떤 선택을 해야 할까요?
```

**Vrew 설정:**
- AI 음성: 메인 나레이터 목소리로 복귀
- 톤: 진지하면서도 희망적

### 섹션 4: 타이틀 (2:30~3:00)

```
[자막]
AI와 함께 찾는 나의 진로
15일 만에 미래를 설계하다
```

**Vrew 설정:**
- 타이틀 자막: 크게, 중앙 배치
- 부제: 메인 타이틀 아래 작게
- 배경: 깔끔한 단색 또는 그라데이션
- 효과: 타이틀 등장 애니메이션

## 4. Vrew 상세 작업 순서

### 4.1 텍스트 입력
1. 위 대사를 복사
2. Vrew `텍스트로 비디오 만들기`에 붙여넣기
3. 섹션별로 구분선(`---`) 사용 권장

### 4.2 AI 음성 설정
1. 각 클립 선택
2. 우측 `AI 음성` 패널
3. 목소리 선택:
   - **메인 나레이터:** 차분한 남성 (예: 준호, 민수)
   - **학부형:** 중년 여성 (예: 수진)
   - **취준생:** 젊은 성인 (예: 지현, 서준)
4. 속도: 0.95 (약간 느리게)
5. `음성 생성` 클릭

### 4.3 배경 이미지/영상 설정
1. 클립 선택
2. 좌측 `미디어` 탭
3. `AI 이미지` 또는 `스톡` 선택
4. 키워드 검색:
   - "technology future"
   - "AI artificial intelligence"
   - "job interview"
   - "statistics graph"

### 4.4 자막 스타일 조정
1. 클립 선택 → 자막 클릭
2. 우측 `스타일` 패널:
   - 폰트: 나눔고딕 Bold 또는 Pretendard
   - 크기: 48~60pt
   - 색상: 흰색 + 검정 외곽선
   - 위치: 하단 중앙

### 4.5 전환 효과
1. 클립 사이 `+` 클릭
2. 전환 효과 선택:
   - 페이드: 부드러운 장면 전환
   - 슬라이드: 섹션 변경 시

   - [ ] 자막이 잘 읽히는가?
