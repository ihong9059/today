## 1. 개요

### 1.1 시스템 정의

**LPR(License Plate Recognition)** 또는 **ANPR(Automatic Number Plate Recognition)** 시스템은 카메라와 AI 기반 영상 처리 기술을 활용하여 차량 번호판을 자동으로 인식하고, 차량의 위치를 추적하여 중앙관제시스템에서 통합 관리하는 스마트 주차 솔루션입니다.

### 1.2 도입 배경

| 도입 요인 | 설명 |
|-----------|------|
| **대형 주차장 증가** | 백화점, 대형마트, 아파트 단지의 지하주차장 규모 확대 |
| **주차 효율성 요구** | 빈 주차면 찾기, 내 차 위치 찾기 등 사용자 편의 향상 |
| **보안 강화** | 도난 차량 감시, 불법 주정차 단속 |
| **무인화 트렌드** | 인건비 절감 및 24시간 운영 필요 |

### 1.3 시스템 핵심 목표

```
┌─────────────────────────────────────────────────────────────┐
│                    시스템 핵심 목표                           │
├─────────────────────────────────────────────────────────────┤
│  1. 차량 번호 자동 인식 및 데이터베이스 저장                    │
│  2. 실시간 차량 위치 추적 및 표시                              │
│  3. 중앙관제센터에서 전체 주차장 통합 모니터링                   │
│  4. 사용자에게 차량 위치 검색 서비스 제공                       │
│  5. 주차 요금 자동 정산                                       │
└─────────────────────────────────────────────────────────────┘
```

## 3. 핵심 기술 요소

### 3.1 차량번호 인식 기술 (LPR/ANPR)

#### 3.1.1 AI 기반 OCR 기술

| 기술 | 역할 | 정확도 |
|------|------|--------|
| **YOLO (You Only Look Once)** | 실시간 번호판 영역 검출 | 빠른 처리 속도 |
| **CNN (Convolutional Neural Network)** | 번호판 문자 인식 | 97.5% 이상 |
| **CRNN (CNN + RNN)** | 한글/숫자 텍스트 추출 | 한글 인식 최적화 |

#### 3.1.2 인식 프로세스

#### 3.1.3 기술 사양

| 항목 | 사양 |
|------|------|
| 인식률 | 99% 이상 |
| 인식 시간 | 0.2초 이내 |
| 동시 인식 | 1~16대 차량 동시 처리 |
| 지원 번호판 | 한국 전 종류 (신형/구형/전기차/외교 등) |
| 야간 인식 | 적외선 LED 조명 지원 |

```sql
CREATE TABLE vehicle_records (

    plate_number    VARCHAR(20) NOT NULL,

    exit_time       DATETIME,
    parking_zone    VARCHAR(10),
    parking_floor   VARCHAR(10),
    parking_spot    VARCHAR(10),
    vehicle_image   BLOB,
    status          ENUM('parked', 'exited')

    spot_id         VARCHAR(20) PRIMARY KEY,
    zone            VARCHAR(10),
    floor           VARCHAR(10),
    is_occupied     BOOLEAN,
    vehicle_plate   VARCHAR(20),
    last_updated    DATETIME
);
```

```
│  → 요금 표시: "주차시간 2시간 30분 / 요금 5,000원"          │
│  → 결제 후 차단기 자동 개방                                  │
│                                                             │
│  무정차 출차 (등록 차량)                                    │
│  ─────────────────                                          │
│  → 월정액/거주자 차량은 자동 인식                           │
│  → 별도 정산 없이 차단기 자동 개방                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
