### 프로젝트 B: IoT 기반 장비 원격 모니터링 시스템

#### 개요
절단기/절곡기 가동 현황을 실시간 원격 모니터링하는 시스템 구축

#### 개발 내용

| 구분 | 내용 | 담당 | 기술 |
|------|------|------|------|
| **IoT 게이트웨이** | 장비 데이터 수집 장치 | 홍광선 | LoRaWAN, Zigbee |
| **센서 연동** | 가동/정지/알람 상태 감지 | 홍광선 | BLE Mesh |
| **클라우드 연동** | 데이터 전송 및 저장 | 홍광선 | TCP/IP |
| **H/W 인터페이스** | 컨트롤러 연동 보드 | 임호균 | Modbus, RS-422 |
| **대시보드** | 웹 기반 모니터링 화면 | 공동 | - |

#### 시스템 구성도
```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  절단기/절곡기 │───▶│ IoT Gateway │───▶│ Cloud Server│
│  컨트롤러    │    │ (LoRa/BLE)  │    │             │
└─────────────┘    └─────────────┘    └──────┬──────┘
                                              │
                   ┌──────────────────────────┼──────────────────────────┐
                   ▼                          ▼                          ▼
            ┌─────────────┐           ┌─────────────┐           ┌─────────────┐
            │ 웹 대시보드  │           │ 모바일 앱   │           │ 알림 시스템  │
            └─────────────┘           └─────────────┘           └─────────────┘
```

#### 주요 기능
1. **실시간 가동 현황** - 장비별 On/Off, 가동률 표시
2. **생산 데이터 수집** - 절단/절곡 횟수, 작업 시간 기록
3. **이상 알림** - 에러 발생 시 즉시 알림 (SMS/푸시)
4. **리포팅** - 일간/주간/월간 생산 보고서

#### 예상 비용: **4,000만원** (6개월)

### 프로젝트 C: 예지정비 (Predictive Maintenance) 시스템

#### 개요
AI 기반 장비 상태 예측으로 고장 전 사전 정비 실현

#### 개발 내용

| 구분 | 내용 | 담당 | 기술 |
|------|------|------|------|
| **센서 설치** | 진동/온도/전류 센서 | 홍광선 | IoT Sensor |
| **데이터 수집** | Edge Device 개발 | 임호균 | ARM, Embedded |
| **신호 처리** | 센서 데이터 전처리 | 임호균 | DSP |
| **AI 분석** | 이상 패턴 감지 모델 | 홍광선 | AI Solution |
| **알림/리포팅** | 정비 권고 시스템 | 공동 | - |

#### 핵심 기능
```
[데이터 수집] → [신호 처리] → [AI 분석] → [정비 권고]
     │              │              │              │
  진동/온도       필터링        패턴 인식      알림 발송
  전류 센서       FFT 분석      이상 감지      리포트 생성
```

1. **실시간 상태 모니터링** - 진동, 온도, 전류 추이 감시
2. **이상 징후 조기 감지** - AI 기반 고장 예측
3. **최적 정비 시점 산출** - 다운타임 최소화
4. **부품 수명 관리** - 교체 주기 자동 알림

#### 기대 효과
- 비계획 다운타임 70% 감소
