## 트러블슈팅

### Vrew 로그인 문제 (Google 계정)
- **증상**: 앱 내에서 Google 로그인이 안 됨 (브라우저에서는 정상)
- **해결 방법**:
  1. 기본 브라우저 확인 (Safari/Chrome)
  2. Vrew 앱 완전 종료 후 재시작 (Cmd + Q)
  3. 캐시 삭제: `~/Library/Application Support/Vrew` 폴더 삭제
  4. 다른 로그인 방법 시도 (이메일 또는 Apple 계정)
  5. 방화벽 설정 확인: 시스템 환경설정 → 보안 및 개인 정보 보호 → 방화벽
  6. Vrew 재설치: https://vrew.voyagerx.com

## Vrew vs Premiere Pro 비교

| 항목 | Vrew | Premiere Pro |
|------|------|--------------|
| 제작사 | 보이저엑스 (한국) | 어도비 (미국) |
| 주요 목적 | AI 기반 자막/음성 편집 | 전문 영상 편집 |
| 대상 | 초보자, 유튜버 | 전문 편집자, 방송/영화 |
| 가격 | 무료 (유료 플랜 있음) | 월 약 24,000원 (구독) |
| 학습 난이도 | 쉬움 (1-2시간) | 어려움 (수개월) |

### 용도별 추천
- **Vrew**: 유튜브 자막 영상, 강의/발표 영상, 숏폼 콘텐츠
- **Premiere**: 뮤직비디오, 광고/상업 영상, 복잡한 효과/합성
